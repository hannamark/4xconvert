Requirements:
  * Maven 2.0.10 or a later version of 2.0.x (not compatible with 2.1.0+)
  
Prior to running an install, you must have a po db admin role. Default name is poadmin. Must have full privs. 
Refer to install/upgrade.properties for default password.
  
Modifying default properties
  * Create local.properties to overwrite any properties in install/upgrade.properties
