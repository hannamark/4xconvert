Looking up Diseases/Conditions
This section provides instructions for looking up disease/condition names to use as Ad Hoc report search criteria.
How to Look Up Diseases/Conditions
1.
To search for the name of the disease or condition, click Look Up.
The Search Disease window appears.
2.
In the Disease Name field, type part or all of the disease/condition that you want to use as your search criterion.
Type as many letters of the disease/condition name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the series of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.
Examples:
To search for “colon cancer”, type col. The system treats this as if you had searched for all diseases that contain “col” anywhere in the name.
To search for “familial Hodgkin lymphoma”, type fam* lymph. The system treats this as if you searched for all diseases that contain “fam” AND “lymph” in the name.
3.
To include synonyms in your search in addition to preferred names, select the Include Synonym check box.
4.
To limit your search to exact matches only, select the Exact Match Only check box.
5.
Click Search.
The system searches for the disease/condition from both preferred names and synonyms in the database, and returns a list of results.
Note:
The system will not return any results if you have typed too few charac­ters in the Disease Name field or have typed a disease/condition that has numerous sub-concepts.
6.
Scroll through the list (if needed) to locate the disease/condition being studied.
Some records in the results list may not be suitable for selection because they are not high enough in the concept hierarchy.
7.
Locate the disease/condition in the list, and click Select.
The drug/condition appears in the Ad Hoc report Disease/Condition search field.
