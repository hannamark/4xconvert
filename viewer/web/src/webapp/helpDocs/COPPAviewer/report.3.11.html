Running Ad Hoc Reports
Ad Hoc reports list all interventional trials in the CTRP system that match criteria that you select from a pre-defined set of search criteria.
For those trials that have reached the processing status Accepted and beyond, the report also provides access to the Trial Summary Report (TSR).
How to Run Ad Hoc Reports
1.
On the navigation pane, click Ad Hoc.
The Ad Hoc Report Type page appears.
2.
Type or select any of the search criteria listed in the following table as appropriate.
Search Criteria
Description/Instructions
Official Title
Type any portion of the trial title using wildcards, or type it exactly as it appears in the official title.
Primary Purpose
Select the primary purpose of the trial from the drop-down list. For primary purpose definitions, see Primary Purpose Values.
Trial Phase
Select the trial phase from the drop-down list. For trial phase definitions, see Trial Phase Values.
Identifier Type
Select the type of identifier you want to search for from the drop-down list. For identifier type descriptions, see Trial Identifier Types.
Identifier
Type all or part of the numeric or alphanumeric identifier for the type of identifier you selected.
Examples:
•
NCI-2000-00015
•
ECOG-1234
Note:
Inter-Group trials use the lead Group’s trial number.
Principal Investigator
Select the principal investigator from the drop-down list.
To narrow the list of investigators displayed, click the Principal Investigator field and type the initial letter of the last name. You can also use the Up and Down arrow keys on your keyboard to scroll through the list.
Processing Status
Select the processing status from the drop-down list. For processing status definitions, see Processing Statuses.
Intervention Type
Select the intervention type from the drop-down list. For valid values, see Intervention Types.
Current Trial Status
Select the current trial status from the drop-down list. For trial status definitions, see Trial Statuses.
Lead Organization
Select the lead organization from the drop-down list.
To narrow the list of organizations displayed, click the Lead Organization field and type the initial letter of the organization’s name. You can also use the Up and Down arrow keys on your keyboard to scroll through the list.
Summary 4 Sponsor
Select the Summary 4 Sponsor from the drop-down list.
To narrow the list of organizations displayed, click the Summary 4 Sponsor field and type the initial letter of the organization’s name. You can also use the Up and Down arrow keys on your keyboard to scroll through the list.
Disease Condition
Click Lookup, and search for the appropriate disease/condition. For detailed instructions, see Appendix B, Looking up Diseases/Conditions.
Search by Submission Type
Select the submission type from the drop-down list. Valid values are as follows:
•
Both. Original and amended submissions
•
Original. Original submissions (indicated by “O” in the results list)
•
Update. Updated submissions (indicated by “U” in the results list)
•
Amendment. Amended submissions (indicated by “A” in the results list)
Search By Trial Category
Select the trial category from the drop-down list. For trial category definitions, see Trial Categories
Summary 4 Funding Category
Select the funding category from the drop-down list. For funding category definitions, see Summary 4 Funding Category.
Milestone
Select the current milestone from the drop-down list. For individual milestone definitions, see Milestones.
3.
Click Run Report.
The trials that match all criteria you selected are listed in the report table.
