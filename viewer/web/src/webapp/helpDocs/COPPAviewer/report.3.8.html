Running Trial Submitted by Institution Reports
The Trial Submitted by Institution report lists all trials submitted by an institution during a given time frame. It provides an indication of submission type, the total of submitted trials by submission type, and the total number of submissions.
The report lists trial counts by submitting site, and provides the following for each site:
•
Total number trials submitted during the given time frame
•
Number of original submissions
•
Number of amended submissions
The report also provides the counts listed above as totals for all sites combined.
You can filter the query by trial submission type (original, amendment, or both).
Submission totals appear at the bottom of the last page of the report. If you don’t see the totals listed, navigate to the last page of the report and then scroll all the way to the bottom of the page.
How to Run Trial Submitted by Institution Reports
1.
On the navigation pane, click Trial Submitted by Institution Report.
The Trial Submitted by Institution page appears.
2.
Enter start and end dates in the fields provided.
3.
In the Submission Type drop-down list, select the type(s) of trial(s) to include in the report. Submission Type values are Original, Amendment, and Both. The later combines original and amended trials in one report.
4.
If you want to exclude CTEP and DCP trials from your report, clear the CTEP/DCP check box.
5.
In the Institution(s) list, select the institution(s) that you want to include in the report. To select more than one institution, press and hold the Ctrl key as you make your selections. To include all institutions in the report, click All.
6.
Click Run Report.
The results are returned in a table at the bottom of the page.
