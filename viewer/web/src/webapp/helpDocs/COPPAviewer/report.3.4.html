Running Summary of Submission Reports
The Summary of Submission report provides the number of trials that organizations submitted during a given time frame.
The report lists trial counts by submitting site, and provides the following for each site:
•
Total number of trials submitted during the given time frame
•
Number of original submissions
•
Number of amendments submitted
The report also displays totals for each of the submission types listed above for all sites in the report.
