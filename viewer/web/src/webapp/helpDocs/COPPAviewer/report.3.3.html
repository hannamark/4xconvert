Running CTRO Reports
Pre-defined queries in the CTRP Viewer enable you to generate and export statistics that help you monitor and assess the progress of trials through their trial processing life cycles.
The table below lists the main steps for running CTRO reports.
Step
Action
1
Select a trial report type form the navigation pane.
2
Select the start and end dates for the date range.
3
Indicate whether or not you want to include CTEP/DCP-submitted trials in your report. These trials are included by default. This step does not pertain to Summary 4 Type reports.
4
When applicable, indicate whether you want statistics for original trial submissions, amendments, or both.
5
Ensure that you have completed all required fields, indicated by asterisks (*).
6
Run the report.
7
Optionally, export the report data to an Excel spreadsheet.
To run most queries, you are required to select a range of dates as part of the criteria. The default start date is 01/01/2009, and the default end date is the current date.
You can type dates in the fields provided as mm/dd/yyyy, or you can select dates by clicking the calendar icon () and selecting the date from the pop-up calendar. The start and end dates are inclusive.
