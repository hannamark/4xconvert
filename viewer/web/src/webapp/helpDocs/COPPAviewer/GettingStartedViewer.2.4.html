Creating CTRP Viewer Accounts
You must have a user account and individual access rights for each of the different categories of reports. If you already have a user account in the NCI CTRP Registration site or Protocol Abstraction site, you can contact application support for report access. If you do not have a user account, you can create one in the NCI CTRP Registration site and then request access to each of the reports required.
The way in which you create a new account in the CTRP Viewer depends on whether you have a current NIH or caGrid account, as follows:
•
New users who do not have a caGrid or NIH account request caGrid credentials via the NCI CTRP Registration Site. Follow the instructions in Creating New CTRP Viewer Accounts.
•
New users who have a caGrid or NIH account use their current credentials to register for an account via the NCI CTRP Registration Site. Follow the instructions in Creating New CTRP Viewer Accounts Using caGrid or NCI Credentials.
New users must have access to a valid email address to create an account
