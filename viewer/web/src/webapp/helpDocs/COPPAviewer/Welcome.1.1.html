NCI Clinical Trials Reporting Program Viewer Online Help
Welcome to the NCI Clinical Trials Reporting Program Viewer Online Help. To start using the CTRP Viewer, select from the following topics:
•
Getting Started introduces you to the CTRP Viewer and provides instructions for registering for an account and for logging in to the system.
•
CTRO Reporting describes how to run pre-defined queries to generate reports that enable you to monitor the performance of the trial abstraction process. It also provides instructions for exporting report data to Excel spreadsheets.
•
Metadata Definitions defines the metadata associated with trials and provides examples of valid values for trial details.
•
Glossary provides definitions of acronyms, abbreviations, and terminology used in this, and other CTRP application user guides.
To learn more about each topic, browse through the online help or use the table of contents, index, or search features of the help. For more information, see Using the Online Help.
You can also find comprehensive procedures for all NCI Clinical Trials Reporting Program Viewer tasks in the NCI Clinical Trials Reporting Program Viewer User’s Guide.
