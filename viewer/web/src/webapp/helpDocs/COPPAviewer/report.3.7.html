Running Trial Submitted by Date Reports
The Trial Submitted by Date report provides the following dates for each trial submitted during a given date range:
•
Trial submission date
•
Current processing date
•
Current milestone date
The report lists all trials by NCI trial identifier, and provides the following additional information for each of them:
•
Submission type (original or amendment)
•
Name of the organization that submitted the trial
•
Lead organization trial identifier
•
Lead organization name
•
Current processing status and milestone date
•
Current milestone and status date
Submission totals appear at the bottom of the last page of the report. If you don’t see the totals listed, navigate to the last page of the report and then scroll all the way to the bottom of the page.
How to Run Trial Submitted by Date Reports
1.
On the navigation pane, click Trial Submitted by Date Report.
The Trial Submitted by Date Report page appears.
2.
Enter start and end dates in the fields provided.
3.
In the Submission Type drop-down list, select the type(s) of trial(s) to include in the report. Submission Type values are Original, Amendment, and Both. The later combines original and amended trials in one report.
4.
If you want to exclude CTEP and DCP trials from your report, clear the CTEP/DCP check box.
5.
Click Run Report.
The results are returned in a table at the bottom of the page.
