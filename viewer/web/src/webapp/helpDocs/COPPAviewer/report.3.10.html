Running Summary 4 Type Reports
The Summary 4 Type Report provides information regarding interventional trials that assist in preparation of Summary 4-Type reports.
The report provides the following information for each trial, as appropriate:
•
Sponsor
•
Site
•
Protocol ID
•
Program Code
•
Open Date
•
Closed Date
•
Phase
•
Type
•
Title
•
Target (accrual)
•
Accrual Center 12m (accruals per center over the last 12 months)
•
Accrual Center to Date (accruals per center to date)
Results are sorted by Clinical Research Category in tabbed pages:
•
Clinical trials involving an agent or device
•
National
•
Externally Peer-Reviewed
•
Institutional
•
Industrial
•
Clinical trials involving other types of interventions (e.g., behavioral modification, nutritional protocols, etc.)
•
Epidemiologic, outcome, or other observational studies
•
Ancillary or correlative studies associated with a clinical trial and other biological studies using clinical specimens that can be linked to individual patient or participant data
Ηοω το Run Summary 4 Type Reports
1.
On the navigation pane, click Summary 4 Type.
The Summary 4 Report Type page appears.
2.
Select start and end dates in the fields provided.
3.
Do one of the following to select the organization of interest:
•
Select the Find by Org Name option, and then type all or part of the organization name.
You can select the appropriate organization from the list of names that appears in the drop-down list when you begin to type a name in the Organization Name field.
- or -
•
Select the Find by Family Name option, and then select the name of the Family from the drop-down list. Once you have made your selection, the system lists the Family’s organizations, from which you can select the appropriate organization.
4.
Click Run Report.
The results are returned in tabbed pages at the bottom of the page.
5.
To view results from other categories, click the tabs at the top of the report.
