Running Trial Processing Reports
The Trial Processing Report provides trial processing completion times, in numbers of days, for each milestone for a single trial. For each milestone, the report also includes the cumulative number of days that have elapsed from the date that the trial was submitted to the date that the milestone processing was completed.
The report lists trial amendments consecutively, as trial versions, each with its own processing progress.
Progress is reported for the following milestones, when applicable:
•
Submission Received Date
•
Submission Acceptance Date
•
Submission Rejection Date
•
Administrative Processing Start Date
•
Administrative Processing Completed Date
•
Scientific Processing Start Date
•
Scientific Processing Completed Date
•
Ready for QC Date
•
QC Start Date
•
QC Completed Date
•
Trial Summary Report Sent Date
•
Submitter Trial Summary Report Feedback Date
•
Initial Abstraction Verified Date
•
On-going Abstraction Verified Date
A milestone that is completed in less than one day is rounded up to 1 day.
How to Run Trial Processing Reports
1.
On the navigation pane, click Trial Processing Report.
The Trial Processing Report page appears.
2.
Enter the NCI trial identifier for the trial of interest in the field provided.
3.
Click Run Report.
The results are returned in a table at the bottom of the page.
