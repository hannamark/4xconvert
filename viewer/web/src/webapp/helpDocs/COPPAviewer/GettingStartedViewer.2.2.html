About the CTRP Viewer
The CTRP Viewer provides you with a series of pre-defined queries to generate reports on clinical trials registered in the NCI Clinical Trials Reporting Program. You can customize and run the queries to acquire trial submission and processing statistics.
The CTRP Viewer provides the following types of reports:
•
Summary of submissions. See Running Summary of Submission Reports
•
Trial processing. See Running Trial Processing Reports
•
Portfolio average milestone. See Running Portfolio Average Milestone Reports
•
Trials submitted by date. See Running Trials Submitted by Date Reports
•
Trials submitted by institution. See Running Trials Submitted by Institution Reports
•
Current milestone. See Running Current Milestone Reports
•
Summary 4 Type Reports. See Running Summary 4 Type Reports
CTRP Viewer supports Internet Explorer 7 (IE 7.0.5730.13 and 7.0.5730.11), IE 8 (8.0.6001.18702), and FireFox 3 (3.0.10) browsers.
