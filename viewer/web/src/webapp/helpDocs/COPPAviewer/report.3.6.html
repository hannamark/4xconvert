Running Portfolio Average Milestone Reports
The Portfolio Average Milestone Report provides all trial milestone completion times, in numbers of days, for trials with reported processing activity during a given time frame.
The table below describes the value of each of the milestone completion statistics for all trials in a selected date range.
Report Heading
Statistics for Each Milestone
Order
Order of occurrence of the particular milestone, sorted by the time it was recorded. For example, if the QC Start Date milestone was recorded three different times, the first one recorded is first in order (1), the second one recorded is second in order (2), and so forth.
1 day or less
Number of trial submissions for which the milestone completion times were one day or less
2 - 10 days
Number of trial submissions for which the milestone completion times were two to ten days, inclusive.
> 10 days
Number of trial submissions for which the milestone completion times were more than ten days
Avg
Average completion time in days
Low
Least number of days for milestone completion
High
Greatest number of days for milestone completion
You can filter the query by trial submission type (original, amendment, or both).
How to Run Portfolio Average Milestone Reports
1.
On the navigation pane, click Portfolio Average Milestone Report.
The Portfolio Average Milestone Report page appears.
2.
Enter start and end dates in the fields provided.
3.
In the Submission Type drop-down list, select the type(s) of trial(s) to include in the report. Submission Type values are Original, Amendment, and Both. The later combines original and amended trials in one report.
4.
If you want to exclude CTEP and DCP trials from your report, clear the CTEP/DCP check box.
5.
Click Run Report.
The results are returned in a table at the bottom of the page.
