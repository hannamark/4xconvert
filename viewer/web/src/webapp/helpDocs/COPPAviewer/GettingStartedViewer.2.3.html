What’s New in This Release
This version of the CTRP Viewer has the following new features:
•
Ad Hoc Reports. You can now create your own reports on all interventional trials in the CTRP system from a pre-defined set of search criteria.
•
Organization search by Family. In the Summary 4 Type report, you can now select the name of an Organization from the Family it belongs to.
