Running Current Milestone Reports
The Current Milestone Report displays the most current milestone for active trials submitted within a given time frame.
The report provides the following information for each trial in addition to the current milestone and milestone date:
•
NCI trial identifier
•
Submission type
•
Lead organization trial identifier
•
Lead organization name
•
Submission date
•
Current processing status
•
Current processing status date
The following counts are included in the report:
•
Total number of trials submitted during the given time frame
•
Number of original submissions
•
Number of amended submissions
Submission totals appear at the bottom of the last page of the report. If you don’t see the totals listed, navigate to the last page of the report and then scroll all the way to the bottom of the page.
How to Run Current Milestone Reports
1.
On the navigation pane, click Current Milestone.
The Current Milestone Report page appears.
2.
Enter start and end dates in the fields provided.
3.
If you want to exclude CTEP and DCP trials from your report, clear the CTEP/DCP check box.
4.
Click Run Report.
The results are returned in a table at the bottom of the page.
