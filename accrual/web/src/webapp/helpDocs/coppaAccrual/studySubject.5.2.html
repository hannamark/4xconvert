Searching for Study Subjects
Study subject records sorted by Study Subject ID.
System should only display those study subject records whose Record Status is in “Pending” or “Active”.
User should be able to search study subject record(s) using the following four criteria: Study Subject ID, Participating Site, Record Status and/or Birth Date. If the user does not input any element, the system will return all the study subject records for the study.
User is not allowed to add, delete or modify a study subject record if an open submission does not exist.
 
How to Search for Study Subjects
1.
Navigate to the trial you want to work with by following the instructions in Selecting Listed Trials on page 15 or Searching for Trials on page 16, and then click the corresponding NCI Trial Identifier link.
2.
On the Navigation pane, click Study Subject Search.
The Search Study Subject page appears.
3.
Type or select your search criteria from one or more of the fields provided.
4.
Click Search.
Your search results are displayed in the List of Study Subject section of the page.
