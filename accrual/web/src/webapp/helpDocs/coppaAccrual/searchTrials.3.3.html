Searching for Trials
You can retrieve existing trials through the CTRP Accrual once you have registered for an account. See Creating an Account on page 8.
The list of trials for which accrual information has been submitted is displayed automatically when you log in to the CTRP Accrual. For information on navigating and working with lists of trials, see Navigating Trial Lists.
How to Search For Trials
1.
On the navigation pane on the left side of the page, click Search Trials.
The Search Trials page appears.
2.
On the Search Trials page, type the NCI Trial Number, NCT Number, or Official Title in the fields provided, then click Search.
The Search Trials page refreshes and displays the trial of interest. For more information on navigating and working with search results, see Navigating Trial Lists.
3.
To view a given trial, click the corresponding NCI Trial Identifier link.
The Accrual Submissions page appears.
