Adding Surgery Records
You must create and select a Treatment Regimen and Cycle record in order to submit intervention details. Refer to Adding Treatment Regimen Records and Adding Cycle Records for instructions.
How to Add Surgery Data
1.
On the navigation pane, under Treatment, click Surgery.
If the Surgery menu is not displayed on the navigation pane, first select the Treatment Regimen menu and select a Treatment Regimen record. Then select a Cycle record.
The Surgery page appears.
2.
Click Add.
The Add Surgery page appears.
3.
To search for the name of the surgical procedure, next to the Surgery Name field, click Look Up.
The Surgery Name window appears.
4.
In the Surgery Name field, type part or all of the intervention name.
Type as many letters of the intervention name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the combination of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.
5.
To include synonyms in your search, select the Include Synonym check box.
6.
To limit your search to exact matches only, select the Exact Match Only check box.
7.
Click Search.
The system returns a list of results
8.
Scroll through the list to locate the intervention, and click Select.
9.
In the Surgery Date field, enter the date the surgery was performed in the format mm/dd/yyyy, or, click the calendar icon, () and select the date from the calendar.
10.
Optionally, in the Additional Surgery Related Information field, provide any other relevant information about the surgical procedure.
11.
Click Save.
