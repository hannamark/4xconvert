Updating Radiation Records
The NCI Outcomes Portal enables you to edit/update existing records.
How to Update Radiation Data
1.
On the navigation pane, under Treatment, click Radiation.
If the Radiation menu is not displayed on the navigation pane, first select the Treatment Regimen menu and select a Treatment Regimen record. Then select a Cycle record.
The Radiation page appears.
2.
In the Edit column for the appropriate record, click the Edit icon () and follow Steps 3 through 4 in Submitting Radiation Data.
