Adding Prior Therapies
 
You are required to complete all elements marked with an asterisk (*).
How to Add Prior Therapy Data
1.
On the navigation pane, click Prior Therapies.
The Prior Therapies page appears. One or more prior therapy records may be listed.
2.
Select the Has the Patient Had Any Prior Therapies check box.
The Prior Therapies page displays a blank prior therapy record.
3.
From the Prior Therapy Type drop-down list, select the type of therapy the patient was given.
4.
In the Prior Therapy Description field, type a description of the type of therapy the patient was given.
5.
In the Action column, click Add.
The Prior Therapies page displays a the new therapy record.
6.
To save all prior therapy records displayed, click Save.
7.
To delete a prior therapy record, in the Action column, click Delete.
