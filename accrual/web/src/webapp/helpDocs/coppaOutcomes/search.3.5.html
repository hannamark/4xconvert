Viewing Existing Patient Data
The system lists the results of patient searches on the Search Patient page in ascending order by patient ID.
Ηοω το View Existing Patient Data
1.
In the Patient ID column, click the patient’s ID hypertext link.
The View Patient page displays the following patient data:
•
Patient ID
•
Birth Date
•
Gender
•
Race
•
Ethnicity
•
Country (of origin)
•
Method of payment
2.
To return to the list of patients, click Back.
