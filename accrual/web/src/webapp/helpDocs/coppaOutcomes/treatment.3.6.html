Submitting Cycle Data
Cycles are associated with a set of interventions. Each patient may have more than one treatment records, and more than one cycle of treatment. For example in the first cycle, a patient may undergo chemotherapy treatment. After tumor assessment, the doctor may decide to order a second cycle with a different dose.
Cycle data includes a cycle name and the date on which the cycle started. Once submitted, this the cycle name is displayed at the top of each page
You must create a Treatment Regimen before you can submit Cycle information. Refer to Adding Treatment Regimens for instructions.
