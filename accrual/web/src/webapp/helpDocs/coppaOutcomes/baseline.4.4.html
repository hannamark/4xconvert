Submitting Staging Data
You are required to complete all elements marked with an asterisk (*).
How to Submit Staging Data
1.
On the navigation pane, click Staging.
The Staging page appears.
2.
From the Staging Method drop-down list, select Clinical or Pathological.
3.
In the T, N, and M fields, type the individual TNM system stage codes. Valid TNM stage codes are as follows:
•
AI (T1a/N0/M0)
•
IB (T1b/N0/M0)
•
IIB (T2b/N0/M0)
•
IIC (T2c/N0/M0)
•
IIIB (T3b/N0/M0
•
IIIC (T3c/N0/M0)
•
IV (Any T/Any N/M1)
4.
In the Stage field, type TNM stage, for example, T3c/N0/M0.
5.
From the Staging System drop-down list, select AICC.
Note:
AICC (Access to Integrated Cancer Care) is the only scheme supported in this version of Outcomes. Later versions will include the SEER (Sur­veillance, Epidemiology, and End Results) system.
6.
Do one of the following:
•
To save the patient staging record without adding tumor marker information, click Save.
- or -
•
To add a tumor marker to the record, follow the instructions in Adding Tumor Markers.
•
click Add Tumor Marker, and continue to the next step.
7.
In the Tumor Marker window, type one or more of the first characters of the tumor marker name.
8.
Click Search.
9.
The Search page refreshes and displays a list of search results.
10.
Navigate to the appropriate tumor marker, and click Select.
The Add Tumor Marker page displays the marker you selected.
11.
In the Tumor Marker Value field, type the appropriate marker value (number).
12.
To search for the tumor marker unit of measure (UOM), next to the Tumor Marker Value field, click Look Up.
The Tumor Marker Value UOM window appears.
13.
In the UOM field, type part or all of the unit of measure, and then click Search.
The Tumor Marker Value UOM window displays the results of your search.
14.
Navigate to the appropriate UOM, and then click Select.
The Add Tumor Marker page displays the value you selected.
15.
Click Save.
The tumor marker record is displayed on the Staging page.
