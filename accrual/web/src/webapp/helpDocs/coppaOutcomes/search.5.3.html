Searching For Patients
Once you have registered for an account, you can retrieve existing patient records that contain demographic, diagnosis, treatment, and outcome data.
Note:
The system displays only those patient records that you have submitted personally.
How to Search for Patients
1.
On the navigation pane on the left side of the page, click Patient Search.
The Search Patient page appears.
2.
In the Patient ID field, type the de-identified patient identification number.
3.
In the Patient Birth Date field, type the patientâ€™s birthday in the format mm/yyyy (e.g., 12/1978).
To display a list of all patients that have been submitted, leave all fields blank and click Search.
The Search Patient page refreshes and displays a list of search results. For more information about search results, see Working with Search Results on page 7.
4.
To view the patient data, click the appropriate link in the Patient ID column.
The VIew Patient page appears.
