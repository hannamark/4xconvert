Adding Drug/Biologic Data
You must create a Treatment Regimen and submit Cycle information in order to submit intervention details. Refer to Adding Treatment Regimens and Submitting Cycle Data for instructions.
How to Add Drug/Biologics Data
1.
On the navigation pane, under Treatment, click Drugs/Biologics.
The Drugs/BIologics page appears.
2.
Click Add.
The Add Drug/Biologic page appears.
3.
To search the PDQ/NCI EVS database for the name of the drug, next to the Drug Name field, click Look Up.
The Search Intervention window appears.
4.
In the Drug/Biologic Name field, type part or all of the intervention name.
Type as many letters of the intervention name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the combination of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.
5.
To include synonyms in your search in addition to preferred names, select the Include Synonym check box.
6.
To limit your search to exact matches only, select the Exact Match Only check box.
7.
Click Search.
The system returns a list of results
8.
Scroll through the list to locate the intervention, and click Select.
9.
In the Dosage Information section, type, select, or look up information in the fields provided as described in the table below. You are required to complete all fields marked with an asterisk (*).
