Adding New Patients
If your search for a patient did not return any results, you can add a new patient record.
How to Add New Patients
1.
On the navigation pane on the left side of the page, click Patient Search.
The Search Patient page appears.
2.
Click Add New Patient.
The Add Patient page appears.
3.
In the Patient ID field, type the de-identified patient identification number.
4.
In the Date of Birth field, type the patient’s birthday in the format mm/yyyy (e.g., 12/1978).
5.
From the Gender drop-down list, select the patient’s gender. If you do not know the patient’s gender, select Unknown.
6.
From the Race list, select the patient’s race. You can select more than one race.
To select multiple races, press and hold the Ctrl key as you click each race.
7.
From the Ethnicity drop-down list, select the patient’s ethnicity.
8.
From the Country of Residence drop-down list, select the patient’s country of origin.
9.
If the patient’s country of origin is the United States, from the Method of Payment drop-down list, select the means by which the patient payed for treatment.
10.
Click Save.
