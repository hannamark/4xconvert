Submitting Diagnosis Data
You are required to complete all fields marked with an asterisk (*).
Ηοω το Submit Diagnosis Data
1.
On the navigation pane, click Diagnosis.
The Diagnosis page appears.
2.
To search for a diagnosis, click Look Up.
The Diagnosis search window appears.
3.
In the Disease Name field, type part or all of the patient’s disease/condition.
Type as many letters of the disease/condition name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the series of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.
4.
To include synonyms in your search in addition to preferred names, select the Include Synonym check box.
5.
To limit your search to exact matches only, select the Exact Match Only check box.
6.
Click Search.
The system searches for the disease/condition from the database, and returns a list of results.
Note:
The system will not return any results if you have typed too few charac­ters in the Disease Name field or have typed a disease/condition that has numerous sub-concepts.
7.
Scroll through the list (if needed) to locate the patient’s disease/condition.
8.
Navigate to the appropriate disease in the list, and click Select.
9.
In the Diagnosis Date field, type the date on which the patient was diagnosed with the disease, using the mm/dd/yyyy format, or, click the calendar icon () and select the date from the calendar.
10.
Click Save.
The record is added and displayed on the Diagnosis page.
