About Baseline Data
Baseline records include the following information about a patient’s treatment:
•
Diagnosis – Diagnosis and date of diagnosis. See Submitting Diagnosis Data
•
Staging – Staging method, THM categorization, and staging system. See Submitting Staging Data
•
Pathology – Pathology grade, grade system, and a description of the pathology. See Submitting Pathology Data
•
Prior Therapies – Prior therapy type and description. See Submitting Prior Therapies Data
•
Performance Status – Performance scores as defined by either the ECOG, Karnofsky, or Lansky. See Submitting Performance Status Data
