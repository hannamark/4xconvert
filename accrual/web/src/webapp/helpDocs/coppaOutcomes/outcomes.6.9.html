Submitting Death Information
 
Ηοω το Sumbit Death Information
1.
On the navigation pane under Patient Outcomes, click Death Information.
The Death Information page appears.
2.
Type or select information in the fields provided as described in the table below. You are required to complete all fields marked with an asterisk (*).
Field Name
Instructions
Cause of Death
Select the cause from the drop-down list. Valid values are as follows:
•
Drug Related
•
Infection
•
New Primary
•
Other
•
Tumor
•
Unknown
Death Date
Enter the date in the format mm/dd/yyyy, or, click the calendar icon, () and select the date from the calendar.
Lesion SIte
1.
Click Look Up.
2.
In the Lesion Site window, type one or more characters in the anatomic site, and click Search.
3.
Navigate to the appropriate record in the search results list, and click Select.
Autopsy Performed?
Select the answer from the drop-down list. Valid values are as follows:
•
Yes
•
No
•
Unknown
Death Cause as Determined by Autopsy
If an autopsy was performed, select the cause of death from the drop-down list. Valid values are as follows:
•
Current disease
•
Protocol treatment
•
Other cause
