Overview of the
The is a web application that provides user access to a data repository containing de-identified patient outcome data for cancer patients. Patient outcome records can contain de-identified demographic, diagnosis, treatment, and outcome data.
Using a common set of patient clinical metadata, medical oncologists/clinicians and administrators at clinicians' offices can accomplish the following objectives:
•
Create (submit) new patient outcome records
•
Update and remove existing patient outcome records
•
Search for patient outcomes outcome records
Note:
Submitters must register as users to gain access to the application features. See Creating an Account Submitters are desig­nated owners of the patient records they submit.
v.1.0 accepts data for patients with solid tumor cancers who have under­gone, or are currently undergoing, interventional treatment.
supports Internet Explorer 7 (IE 7.0.5730.13 and 7.0.5730.11), IE 8 (8.0.6001.18702), and FireFox 3 (3.0.10) browsers.
