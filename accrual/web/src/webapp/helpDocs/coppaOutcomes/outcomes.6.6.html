Adding Lesion Assessment Records
The NCI Outcomes Portal enables you to create new outcome records.
Ηοω το Add Lesion Assessmet Records
1.
On the navigation pane under Patient Outcomes, click Lesion Assessment.
The Lesion Assessment page appears. Existing lesion records are displayed if applicable.
2.
Click Add.
The Add Lesion Assessment page appears.
3.
Type and/or select information in the fields provided as described in the table below. You are required to complete all fields marked with an asterisk (*).
Field Name
Instructions
Treatment Plan Name
Select the name from the drop-down list.
Lesion Number
Type the number in the field provided.
Lesion SIte
1.
Click Look Up.
2.
In the Lesion Site window, type one or more characters in the Anatomic Site field, and click Search.
3.
Navigate to the appropriate record in the search results list, and click Select.

Enter the date in the format mm/dd/yyyy, or, click the calendar icon, () and select the date from the calendar.
