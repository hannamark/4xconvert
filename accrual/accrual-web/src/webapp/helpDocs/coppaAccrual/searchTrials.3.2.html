Selecting Listed Trials
The CTRP Accrual site displays the Trial Search/List of Trials page automatically after you have logged in and accepted the notification regarding respondent burden.
This page list trials for which you are authorized to submit accrual records, and which meet the following criteria:
•
Current Trial Status is “Active” or a later status
•
Current Processing Status is “Abstracted, “Abstracted Verified Response / No Response”, or later (with the exception of amendments)
The List of Trials page displays the trials to which you have been granted access. They are listed by NCI Trial Identifier in ascending order by default.
You can sort the list of trials by clicking a column heading. To reverse the sort order, click the column heading again.
How to Select Trials on the List of Trials Page
1.
Navigate to the trial you want to work with by following the instructions in Navigating Trial Lists.
2.
To view a given trial, click its corresponding NCI Trial Identifier link.
The Accrual Submissions page appears.
