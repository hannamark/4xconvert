Reviewing Accrual Records
CTRP Accrual enables you to review the administrative details associated with a particular trial/study registered with the CTRP.
Each accrual submission record is identified by either of the following statuses:
•
Opened. Accrual records that have been created but not completed or submitted to the CTRO are considered opened. You can add, update, and delete study subjects only if the current submission is opened.
There can be no more than one opened accrual submission record per trial.
•
Submitted. Accrual records that have been completed are considered submitted. You can not add, update, or delete study subjects if all existing accrual submissions are completed.
How to Review Accrual Records
1.
Navigate to the appropriate trial and click the NCI Trial Identifier. For instructions on locating a particular trial, see Searching for Trials.
The Accrual Submissions page appears and displays any submission records that may have been submitted previously, and/or opened but not submitted.
2.
Locate the accrual record you want to review, and click the accrual title in the Label column.
The Accrual Submission Details page displays the current accrual information .
3.
To return to the accrual record, click Back.
