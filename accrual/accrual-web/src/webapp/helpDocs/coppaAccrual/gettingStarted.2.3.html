Requesting an Accrual Site Account
In order to submit accrual records via CTRP Accrual, you must request access from the CTRO (Clinical Trials Reporting Office) and have a valid CTRP Accrual account.
How to Request an Accrual Site Account
1.
Self-register for an account in the NCI CTRP Registration Site by following the instructions in Creating New CTRP Accrual Accounts.
2.
Contact the CTRO to request an account for the Accrual site.You can contact the CTRO via email at ncictro@mail.nih.gov.
You can register for an account via the NCI CTRP Registration Site. The CTRP Accrual site home page provides a link to the Registration Site account page.
The way in which you create a new account in the CTRP Accrual depends on whether you have a current NIH or caGrid account, as follows:
•
New users who do not have a caGrid or NIH account request caGrid credentials via the NCI CTRP Registration Site. Follow the instructions in Creating New CTRP Accrual Accounts.
•
New users who have a caGrid or NIH account use their current credentials to register for an account via the NCI CTRP Registration Site. Follow the instructions in Creating New CTRP Accrual Accounts Using caGrid or NCI Credentials.
New users must have access to a valid email address to create an account
