Adding Study Subjects
You can add one or more new study subjects to opened accrual records for each trial to which you have been granted access.
Study subject records include demographic data as well as the disease name under investigation. CTRP Accrual enables you to search for and select diseases and conditions from a snapshot of the PDQ (Physician Data Query) database and to add them to a study subject’s record. PDQ is an online database developed and maintained by the National Cancer Institute. You can find more specific information about PDQ at http://cancer.gov/cancer_information/pdq/.
Note:
The PDQ terminology snapshot may be extended by new terms vali­dated in EVS (Enterprise Vocabulary Service).
You are required to complete all fields marked with an asterisk (*).
How to Add Study Subject Records
1.
Navigate to the trial you want to work with by following the instructions in Selecting Listed Trials or Searching for Trials, and then click the corresponding NCI Trial Identifier link.
2.
On the Navigation pane, click Study Subject Search.
The Search Study Subject page appears.
3.
Click Add New Study Subject.
The Add Study Subject page appears.
4.
Type or select the appropriate demographic information as described in the table below.
Study Subject Information
Instruction/Description
ID
Type the unique ID as per the lead organization or the study site where the subject is registered.
Birth Date
Type the subjects date of birth in the format mm/yyyy.
Gender
Select the subject’s gender. If gender information is not available, select Unknown.
The system compares the subject's gender value you select with the study’s eligible gender value(s). If the gender you select does not correspond with the study-level eligible gender, the system displays an error.

Not Reported

To select multiple races, select one race, and then press and hold the Ctrl key as you select the other(s).

Hispanic or Latino
•

•
Not Reported
•
Unknown
Country
Select the subject’s country of origin.
Zip Code
Type the subject’s Zip Code if known. This field is mandatory if the Country of origin is United States.
5.
Do one of the following to record the date that the subject was registered for the trial:
•
In the Registration Date field, type the date in the format mm/dd/yyyy.
- or -
•
Click the calendar icon () and select the date from the pop-up calendar.
The registration date must precede the accrual cut-off date.
6.
For United States study subjects only, from the Study Subject Method of Payment drop-down list, select the appropriate payment method.
7.
To assign a disease to the study subject, next to the Disease field, click Look Up and search for the name of the disease or condition being studied.
The Search Diseases window appears.
8.
In the Disease Name field, type part or all of the disease/condition being studied.
9.
Type as many letters of the disease/condition name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the series of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.To include synonyms in your search in addition to preferred names, select the Include Synonym check box.
10.
To limit your search to exact matches only, select the Exact Match Only check box.
11.
Click Search.
The system searches for the disease/condition in the database, and returns a list of results.
Note:
The system will not return any results if you have typed too few charac­ters in the Disease Name field.
12.
Scroll through the list (if necessary) to locate the disease/condition being studied, and click Select.
The system populates the Disease field for you.
13.
From the Participating Site drop-down list, select the appropriate site.
14.
Click Save.
The study subject record appears in the List of Study Subjects.
The Record status remains “Pending” until you submit the associated accrual record.
