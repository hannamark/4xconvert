Searching for Diseases
CTRP Accrual enables you to search for and select diseases and conditions from the PDQ (Physician Data Query) database and to add them to a study subject’s record. PDQ is an online database developed and maintained by the National Cancer Institute. You can find more specific information about PDQ at http://cancer.gov/cancer_information/pdq/.
Ηοω το Search for Diseases
1.
Follow Steps 1 through 3 in Adding Study Subjects.
2.
To assign a disease to the study subject, next to the Disease field, click Look Up and search for the name of the disease or condition being studied.
The Search Diseases window appears.
3.
In the Disease Name field, type part or all of the disease/condition being studied.
Type as many letters of the disease/condition name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the series of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.
4.
To include synonyms in your search in addition to preferred names, select the Include Synonym check box.
5.
To limit your search to exact matches only, select the Exact Match Only check box.
6.
Click Search.
The system searches for the disease/condition in the database, and returns a list of results.
Note:
The system will not return any results if you have typed too few charac­ters in the Disease Name field.
7.
Scroll through the list (if necessary) to locate the disease/condition being studied, and click Select.
The system populates the Disease field for you.
