Submitting Off-Treatment Data
You must create a new Treatment Regimen or select an existing one before you can submit off-treatment data. Refer to Adding Treatment Regimen Records for instructions.
Off-Treatment information includes the date and reason why the patient went off treatment.
How to Submit Off-Treatment Data
1.
On the navigation pane, under Treatment, click Off Treatment.
If the Off Treatment menu is not displayed on the navigation pane, first select the Treatment Regimen menu and select a Treatment Regimen record.
2.
In the Date Off Treatment field, type the date of the patientâ€™s last treatment in the format mm/dd/yyyy, or, click the calendar icon, () and select the date from the calendar.
3.
In the Off Treatment Reason drop-down list, select the reason treatment was ended. Valid values are as follows:
 
Value Description
 
Death
 
Adverse Event/Side Effects/Complications
 
Lost to Followup
 
Refused Followup
 
Followup Completed
 
Other
4.
Click Save.
