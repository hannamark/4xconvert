Adding New Patient Records
The NCI Outcomes Portal enables you to add new patient records. New records appear in your list of patients.
How to Add New Patient Records
1.
On the navigation pane on the left side of the page, click Patient Search.
The Search Patient page appears.
2.
Click Add New Patient.
The Add Patient page appears.
3.
Type or select the appropriate demographic information as described in the table below.
Study Subject Information
Instruction/Description
ID
Type the de-identified patient identification number
Birth Date
Type the subjects date of birth in the format mm/yyyy (e.g., 12/1978)
Gender
Select the subject’s gender. If gender information is not available, select Unknown.
Race
Select one or more values for race:
•
American Indian or Alaska Native
•
Asian
•
Black or African American
•
Native Hawaiian or Other Pacific Islander
•
Not Reported
•
Unknown
•
White
You can select more than one race, except if you include either Not Reported or Unknown.
To select multiple races, select one race, and then press and hold the Ctrl key as you select the other(s).
Ethnicity
Select a value for ethnicity:
•
Hispanic or Latino
•
Not Hispanic or Latino
•
Not Reported
•
Unknown
Country of Residence
Select the patient’s country of origin.
Method of Payment
If the patient’s country of origin is the United States, select the means by which the patient paid for treatment.
4.
Click Save.
