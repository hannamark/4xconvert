Adding Cycle Records
You must create and then select a Treatment Regimen before you can submit Cycle information. Refer to Adding Treatment Regimen Records for instructions.
How to Add Cycle Records
1.
On the navigation pane, under Treatment, click Cycle.
The Cycle page appears.
2.
Click Add.
The Add Cycle page appears.
3.
In the Cycle field, type the name/number or description of the cycle.
If a cycle record already exists, the cycle name may be listed when you click the Cycle field. You can select the name rather than type it.
4.
In the Cycle Start Date field, enter the date the current course started in the format mm/dd/yyyy, or, click the calendar icon, () and select the date from the calendar.
5.
Click Save.
