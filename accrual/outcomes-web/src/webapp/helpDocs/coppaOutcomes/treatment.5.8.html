Updating Cycle Records
The NCI Outcomes Portal enables you to edit/update existing records.
How to Update Cycle Data
1.
On the navigation pane, under Treatment, click Cycle.
If the Cycle menu is not displayed on the navigation pane, select the Treatment Regimen menu and then select a treatment regimen record. Then return to the Cycle page.
The Cycle page appears.
2.
In the Edit column for the appropriate record, click the Edit icon () and follow Steps 3 through 5 in Submitting Cycle Data.
