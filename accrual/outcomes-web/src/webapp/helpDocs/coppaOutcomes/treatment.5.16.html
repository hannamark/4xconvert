Adding Radiation Records
You must create a new Treatment Regimen or select an existing one before you can submit radiation details. Refer to Adding Treatment Regimen Records and Adding Cycle Records for instructions.
How to Add Radiations Data
1.
On the navigation pane, under Treatment, click Radiation.
If the Radiation menu is not displayed on the navigation pane, first select the Treatment Regimen menu and select a Treatment Regimen record. Then select a Cycle record.
The Add Radiation page appears.
2.
Click Add.
The Add Radiation page appears.
3.
In the Dosage Information section, type, select, or look up information in the fields provided as described in the table below. You are required to complete all fields marked with an asterisk (*).
