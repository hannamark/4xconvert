Submitting Cycle Data
Cycles are associated with one or more interventions. Each patient may have more than one treatment regimen record, and more than one cycle of treatment. For example, in the first cycle, a patient may undergo chemotherapy treatment. After tumor assessment, the doctor may decide to order a second cycle with a different dose.
Cycle data includes a cycle name and the date on which the cycle started. Once submitted or selected, the cycle name is displayed at the top of each page
You must create a new Treatment Regimen or select an existing one before you can submit Cycle information. Refer to Adding Treatment Regimen Records for instructions.
