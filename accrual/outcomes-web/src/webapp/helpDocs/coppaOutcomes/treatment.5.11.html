Updating Drug/Biologic Data
The NCI Outcomes Portal enables you to edit/update existing records.
How to Update Drug/Biologic Data
1.
On the navigation pane, under Treatment, click Drug/Biologic.
If the Drug/Biologic menu is not displayed on the navigation pane, select a treatment regimen and cycle record. Then return to the Drug/Biologic page.
The Drug/Biologic page appears.
2.
In the Edit column for the appropriate record, click the Edit icon () and follow Steps 3 through 10 in Submitting Drug/Biologics Data.
