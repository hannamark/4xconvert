Adding Prior Therapy Records
You are required to complete all elements marked with an asterisk (*).
How to Add Prior Therapy Data
1.
If you have not selected a patient record, navigate to the patient record in the List of Patients, and click the Patient ID link or Update icon. See Updating Patient Records for details.
2.
On the navigation pane, click Prior Therapies.
The Prior Therapies page appears. One or more prior therapy records may be listed.
3.
Select the Has the Patient Had Any Prior Therapies check box. This box is selected automatically if any patient therapy records currently exist.
Do not de-select the Has the Patient Had Prior Therapies check box after you have added prior therapy records. Doing so will delete all the patient’s prior therapy records.
The Prior Therapies page displays a blank prior therapy record.
4.
From the Prior Therapy Type drop-down list, select the type of therapy the patient was given.
5.
In the Prior Therapy Description field, type a description of the type of therapy the patient was given.
6.
In the Action column, click Add.
The Prior Therapies page displays the new therapy record.
7.
To delete a prior therapy record, in the Action column, click Delete.
8.
To save all modifications to the patient’s prior therapies, including those that you have added or deleted, click Save.
The system records information only after you have clicked Save.
