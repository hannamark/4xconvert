Viewing Existing Patient Details
The NCI Outcomes Portal system lists the results of patient searches on the Search Patient page in ascending order by patient ID by default.
You can reverse the sort order by clicking the Patient ID column header.
How to View Existing Patient Details
1.
In the Patient ID column, click the patient’s ID hypertext link to activate the patient record.
Once you have activated the patient record, you can record/modify baseline, treatment, and outcome data as nescessary.The View Patient’s Details page displays the following patient data:
•
ID
•
Date of Birth
•
Gender
•
Race
•
Ethnicity
•
Country of Residence
•
Method of Payment
2.
To return to the list of patients, click Back.
