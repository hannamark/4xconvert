Viewing Existing Patient Details
The Outcomes system lists the results of patient searches on the Search Patient page in ascending order by patient ID.
How to View Existing Patient Details
1.
In the Patient ID column, click the patient’s ID hypertext link.
Once you have selected the patient ID, you can modify existing data and record new baseline and outcome information as nescessary.
The View Patient’s Details page displays the following patient data:
•
Patient ID
•
Date of Birth
•
Gender
•
Race
•
Ethnicity
•
Country of Residence
•
Method of Payment
2.
To return to the list of patients, click Back.
