About Treatment Data
Treatment records include the following information about a patient’s treatment:
•
Treatment regimen – Treatment name and description. See Submitting Treatment Regimen Data
•
Cycle – Treatment cycle description and the cycle’s start date. See Submitting Cycle Data
•
Drug/Biologics – Drug/biologic name and dosing information. See Submitting Drug/Biologics Data
•
Surgery – Surgical procedure name and date of surgery. See Submitting Surgery Data
•
Radiation – Type of radiation and administration details. See Submitting Radiation Data
•
Off treatment – Date and reason why the patient went off treatment. See Submitting Off-Treatment Data
Before you begin submitting treatment data, you must record a Treatment Regimen and associated Cycle if you have not already done so. Active treatment and cycles are displayed on the summary of treatment and other Outcomes-related screens. Refer to Adding Treatment Regimen Records and Adding Cycle Records for instructions.
You may modify, but can not delete, any records you create in the Treatment sections.
