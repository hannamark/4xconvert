Abstracting Human Subject Safety Information
The information you are required to provide depends on the Board approval status and other indicators, as noted in the instructions that follow.
Note:
You will need the IRB approval document for a single site to abstract this category of information.
Before you begin to abstract human subject safety information, ensure that the participating sites information has been completed.
Note:
The only field that you are required to complete in this section is the Board Approval Status. Other fields are either optional or non-editable, which are indicated by gray fields).
How to Abstract Human Subject Safety Information
1.
On the navigation pane, click Human Subject Safety.
The Human Subject Safety Information section appears.
2.
In the Board Approval Status drop-down list, select the current state of the trial with respect to Investigational Review Board (IRB) approval. The following table lists valid methods.
Status
Definition
Submitted, Approved
IRB approval has been requested and obtained
Submitted, Exempt
IRB has granted an exemption in response to the approval request
Submission Not Required
Study does not require human subjects review
3.
Indicate whether the submitted IRB Approval is site-related by selecting either Yes or No.
4.
If you selected Submitted, Approved in the Board Approval Status drop-down list, type the number assigned by the human subjects review board upon approval of the protocol in the Board Approval Number field. If the number does not appear in the IRB approval document, type the date of approval using the mm/dd/yyyy format.
5.
If you selected Submitted, Approved or Submitted, Exempt in the Board Approval Status drop-down list, you must select the name of the Board from the Board Name drop-down list. If the name is not listed, search for or register a Board by following the instructions in Chapter 8, Working With Organization Records,.
Note:
If you registered a new organization with the CTRO in the previous step, the system will notify you when the record is added so that you may continue your abstraction.
6.
If you provided a Board name in the previous step, you must also provide the Board’s contact information. If you selected the Board name from the drop-down list or by searching for it, the contact information is displayed automatically in the fields provided.
7.
If you selected Submitted, Approved or Submitted, Exempt in the Board Approval Status drop-down list, you must select the site from which the IRB approval was obtained from the Board Affiliation drop-down list. If the site is not listed, add the participating site to the trial by following the instructions in Abstracting Participating Site Information.
8.
To save the details you have abstracted, click Save.
9.
Do one of the following to continue:
•
To review the data you provided on the previous page, click Back.
- or -
•
To proceed with the abstraction and move to the next page, click Next.
