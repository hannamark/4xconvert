Editing Sub-group Records
You can edit sub-group records as necessary.
How to Edit Sub-group Records
1.
On the navigation pane, click Sub-group.
The Sub-groups page appears.
2.
In the Edit column for a given record, click the Edit icon ().
The Add/Edit Sub-group page appears.
3.
Modify the sub-group details according to instructions in Adding Sub-groups to a Trial.
4.
Click Save.
