Abstracting Sub-Groups
A sub-group is a subset of study subjects within an arm based on factors that may influence the study. These sub-groups are established for the purpose of assessing outcomes. For example, the study population can be grouped by age range, gender, race, location, or condition. Each study must include at least one sub-group dedicated to outcome assessment.
Trials can have multiple sub-group records. Each record displays the following information about the disease or conditions:
Column Name
Definition
Code
Characters, or phrase used to identify the sub-group.
Description
Brief summary of the sub-group outcome factors.
