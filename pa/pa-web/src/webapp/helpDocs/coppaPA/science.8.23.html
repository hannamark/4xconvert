Editing Intervention Records
You can edit the details of an intervention, and reenter or generate its description. However, if you need to change the intervention type or name, you must delete the record and create a new one with the correct information. See Deleting Interventions from a Trial.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Edit Intervention Records
1.
On the navigation pane, click Intervention.
The Interventions page appears.
2.
In the Edit column for a given record, click the Edit icon.
The Edit Intervention page appears.
3.
To edit the intervention description and/or details of drugs, radiation, and procedures/surgeries, follow the instructions in Adding Interventions to a Trial.
4.
Click Save.
