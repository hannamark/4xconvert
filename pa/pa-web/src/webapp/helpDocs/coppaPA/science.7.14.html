Editing Other Eligibility Criteria
You can edit both the default mandatory criteria and other criteria that you or others may have added during abstraction.
EditBefore you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Other Eligibility Criteria
1.
On the navigation pane, click Eligibility Criteria.
2.
The Eligibility Criteria page appears.To edit the required fields only, follow the instructions in Abstracting Trial Eligibility Criteria.
3.
To edit additional criteria, in the Edit column for the record you want to modify, click the Edit icon ().
4.
The Add/Edit Eligibility Criterion page appears, displaying the criterion as submitted. Modify any or all of the information, and then click Save. See Adding Other Eligibility Criteria.
