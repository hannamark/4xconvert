Linking Investigators to a Trial
You must link at least one Participating Site to the trial before abstracting investigator information. See Linking Organizations to a Trial
How to Link an Investigator to a Trial
1.
On the navigation pane, click Participating Sites.
The Participating Sites Page appears.
2.
Click Add.
The Participating Sites tab appears.
Note: You must abstract the Participating Site information in order to proceed to the next step.
3.
Click the Investigators tab.
The Investigators tab displays the trial investigators that may have been added during trial submission or abstraction.
4.
Click Add and search for the investigatorâ€™s name by following the instructions in Looking Up Registered Persons.
5.
When you find the investigator in the search results list, assign the investigator role.
The person's record you selected appears on the Investigators tab.
6.
If you want to indicate that an investigator is the primary contact, click the Set as Site Primary Contact icon () next to this investigator's record.
You will not be able to complete the abstraction if you do not indicate the primary contact.
