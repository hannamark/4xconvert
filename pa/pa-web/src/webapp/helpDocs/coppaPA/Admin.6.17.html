Linking Organizations to Proprietary Trials
You can associate, or link, participating sites with the trial you are abstracting.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Link Organizations to Proprietary Trials:
1.
On the navigation pane, click Participating Sites.
The Participating Sites Page appears.
2.
Click Add.
The Participating Sites Add/Edit Participating Site page appears.
3.
Next to Organization Name, click Look Up, search for the organization, and then click Select in the list of search results.
4.
In the Local Trial Indentifier field, type the appropriate ID.
5.
Next to Site Principal Investigator, click Look Up Person, search for the organization, and then click Select in the list of search results.
6.
Optionally, type the site-specific program code in the field provided.
7.
In the Site Recruitment Status field, select the status from the drop-down list.
8.
Optionally, to view the trialâ€™s recruitment status history, click History.
9.
In the Site Recruitment Status Date field, type the recruitment date (as mm/dd/yyyy), or, click the calendar icon () and select the date from the calendar.
10.
Optionally, in the Date Opened for Accrual and Date Closed for Accrual fields, type the status dates (as mm/dd/yyyy), or, click the calendar icon () and select the dates from the calendar.
If you provide a Closed for Accrual date, you must also provide an Opened for Accrual date.
11.
Click Save.
