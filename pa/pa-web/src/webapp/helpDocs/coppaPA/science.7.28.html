Editing Arm Records
You can edit arm records as necessary.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Edit Arm Records
1.
On the navigation pane, click Arm.
2.
The Arms page appears.In the Edit column for a given record, click the Edit icon ().
3.
The Edit Arm page appears.Modify the arm details according to instructions in Adding Arms to a Trial.
4.
Click Save.
