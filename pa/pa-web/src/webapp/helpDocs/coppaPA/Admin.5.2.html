About Administrative Data
You can edit or add most trial-related administrative information except identifiers that the system generates, such as NCI Trial Identifier.
Administrative data for trials comprises the following information:
•
General trial details
•
NCI-specific information
•
Regulatory information
•
Human subject safety
•
Trial IND/IDE
•
Trial status
•
Trial funding
•
Participating sites
•
Collaborators
•
Trial-related documentation
The navigation pane expands so that you can navigate easily to the specific category of data you would like to add or edit.
The summary remains displayed at the top of each page as you navigate through the Administrative Data section. This serves as a visual reminder of the trial you are abstracting or editing.
