Viewing and Adding Milestones
How to View and Add Milestones
1.
On the navigation sidebar, click Trial Milestones.
The Trial Milestones page appears and displays all milestones recorded to date.
2.
To record a milestone, click Add.
The Add Milestone page appears.
3.
In the Milestone field, select the milestone you want to add to record from the drop-down list. The following table lists valid milestones and their definitions.

4.
In the Date field type the date, or click the calendar icon, and select the milestone date.
5.
In the Comments field, type the action that triggered the milestone, or any comments about the event.
6.
Click Save.
The milestone is displayed on the Trial Milestones page.
7.
Repeat the steps above after you complete each abstraction update as appropriate.
