Abstracting Trial Descriptions
Complete this section for non-proprietary trials.
Trial descriptions include the following information:
•
Brief title
•
Brief summary
Additionally, they may include the following optional information:
•
Detailed description
•
Primary objective
•
Secondary objective
•
Tertiary objective
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Abstract Trial Descriptions
1.
On the navigation pane, click Trial Description.
2.
The Trial Description page appears.In the Brief Title field, type a title that summarizes the purpose of the trial.
3.
In the Brief Summary field, type a summary description of the trial.
4.
Optionally, in the Detailed Description section, type all the information that accurately describes the trial in detail.
The Primary, Secondary, and Tertiary objective fields are read-only.
5.
Click Save.
