Abstracting Trial Descriptions
How to Abstract Trial Descriptions
1.
On the navigation pane, click Trial Description.
2.
The Trial Description page appears.
3.
In the Brief Title field, type a title that summarizes the purpose of the trial.
4.
In the Brief Summary field, type a summary description of the trial.
5.
Optionally, in the Detailed Description field, type all the information that accurately describes the trial in detail.
6.
Click Save.
