Interventions
Constraint/Rule:
1. This use case is applicable to interventional trials only.
2. Interventional study must have at least one arm. This rule is verified at abstraction completion.
3. Single arm must have at least one intervention assigned to it unless arm type is 'No-intervenÂ­tion'. This rule is verified at abstraction completion.
4. No intervention assignment is allowed for 'No intervention' arm.
5. Interventions must be added to trials before they can be assigned to arms.
6. Arm information can be deleted. Deletion confirmation is required before issuing arm deletion.
7. Intervention assignment must be deleted in case of corresponding arm deletion or intervention deletion (referential integrity via FK, cascade deletion).
8. Old intervention assignment must be deleted if arm is edited and interventions assignment has changed.
