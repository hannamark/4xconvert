Validating Trial Status Details
Ensure that the relationships between statuses and dates follow the rules provided in the tables and diagrams that follow.
Trial Status validation does not pertain to abbreviated trials.
Trial status refers to the current stage or state of a clinical trial or study relative to other stages and its ability to enroll participants/patients.
Valid trial statuses used in CTRP are mapped to ClinicalTrials.gov-defined recruitment values as listed in the table below.

The diagram below illustrates the relationships between statuses. Statuses must have been recorded in the order shown. For example, an Administratively Completed status can not be recorded until an Active status has been recorded. There are two exceptions to this rule, as indicated by the curved red lines in the diagram:
•
Transition from In Review to Active. May be accepted if you also record an Approved state with the status date equal to the Active status date.
•
Transition from Active to Closed to Accrual and Intervention. May be accepted if you also record a Closed to Accrual state with the status date equal to the Closed to Accrual and Intervention status date.
Valid dates for a given trial status depend on the other values you have entered, and whether those dates are Actual (current, or past) or Anticipated (future).
The following diagram illustrates these rules. Line A represents the flow of statuses over time. Time between statuses is not important. Line B represents the Trial Start date relative to the statuses, and Line C represents the same for Primary Completion dates. Closely-spaced dashed lines indicate dates that are Anticipated. Widely-spaced dashed lines indicate dates that are Actual.

How to Review Trial Status Details
1.
In the navigation pane, click Trial Status.
The Trial Status page appears.
Trial status details for abbreviated trials contain a different set of elements.
2.
To view trial status information previously recorded, click History.
3.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Items with an asterisk are required fields.
Field Label
Description/Instructions
Current Trial Status*
Select the trial’s accrual status from the drop-down list. For Current Trial Status definitions, see Current Trial Status.
Current Trial Status Date*
Enter the date on which the current trial status became effective. See Selecting and Entering Dates.
Why Study Stopped
If you selected the Administratively Complete, Withdrawn, or Temporarily Closed to Accrual status, type the reason why the study has ended or is not currently accruing.
Trial Start Date*
1.
Enter the date on which the trial started, or is expected to start. See Selecting and Entering Dates and Rules for Status/Dates relationships.
2.
Indicate whether the start date is the one on which the trial is expected to start, or the date on which it actually started by selecting Anticipated or Actual.
Primary Completion Date
1.
Enter the date on which the trial ended, or is expected to end. See Selecting and Entering Dates and Rules for Status/Dates relationships.
2.
Indicate whether the start date is the one on which the trial is expected to start, or the date on which it actually started, by selecting Anticipated or Actual.
4.
If you made any changes, click Save.
