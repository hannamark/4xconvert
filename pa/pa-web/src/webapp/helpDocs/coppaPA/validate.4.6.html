Validating Trial Documents
Protocol and IRB documents are required for all non-proprietary trials. A Proprietary Protocol Template is required for all proprietary trials. Ensure that all required documents have been submitted and that these documents include information required for abstracting trials. For example, the informed consent document must be included in the protocol document or submitted in a separate file. A participating sites document must be submitted for a multi-site trial if the participating sites data is not included in the protocol document.
You can add, edit, or delete documents that are not required. You only can replace the required ones.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Review Trial Documents
1.
In the navigation pane, click Trial Related Documents.
The Trial Documents page appears.
2.
Do one of the following to continue:
•
To add a new document, click Add and follow the instructions provided in Adding Trial-Related Documents.
•
To modify or replace an existing document, click the Edit icon () and follow the instructions provided in Editing and Replacing Documents.
•
To delete a document other than a Protocol or IRB Approval document, click the Delete icon () in the Delete column and follow the instructions provided in Deleting Trial-Related Documents.
There may be more than one record in the table. Be sure you select the appropriate Edit or Delete icon.
3.
Proceed to Accepting and Rejecting Trials.
