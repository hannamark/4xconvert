Viewing and Adding Milestones
Because milestones indicate the progress of a trial, you can view, but not change recorded milestones. However, you can add records throughout the life cycle of the trial submission.
Before you begin abstracting any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data. For instructions, see Checking In and Checking Out Trials.
How to View and Add Milestones
1.
On the navigation pane, click Trial Milestones.
The Trial Milestones page appears and displays all milestones recorded to date.
2.
To record a milestone, click Add.
The Add Milestone page appears.
3.
In the Milestone field, select the milestone you want to add to record from the drop-down list. The following table lists valid milestones, their definitions, processing status, and the rules for recording them.
