Editing Arm Records
You can edit arm records as necessary.
How to Edit Arm Records
1.
On the navigation pane, click Arm.
The Arms page appears.
2.
In the Edit column for a given record, click the Edit icon ().
The Edit Arm page appears.
3.
Modify the arm details according to instructions in Adding Arms to a Trial.
4.
Click Save.
