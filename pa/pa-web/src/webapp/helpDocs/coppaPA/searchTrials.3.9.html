Granting Admin Rights to NCI CTRP Registration Site Users
Each organization registered in the CTRP system can have one primary site administrator (Admin) who is responsible for reviewing and updating user access to trials owned by the organization, and for assigning site administrator status to other users in the organization. In your role as abstractor, you can accept or reject requests for Admin rights from NCI CTRP Registration Site users.
The Inbox lists Admin access requests by default in the Pending Admin User section; you do not have to search for them. You can accept or reject these requests on a per-user basis.
If you search for a trial in the Inbox, only the list of search results will be displayed. To re-display the list of pending Admin users, click the Inbox menu on the navigation pane again.
How to Grant Admin Rights to NCI CTRP Registration Site Users
1.
On the top section of the navigation pane, click Inbox.
2.
The Search Trial page appears. Requests for Site Administration, if any, are listed under Pending Admin User by default.In the First Name column, and click the link for the person to whom you want to accept or reject a request.
3.
All updated trials matching your criterion are listed in the Search Results section.Do one of the following:
•
To accept the request, click Accept.
- or-
•
To reject the request, click Reject.
The request is removed from the list of pending Admins, and the user is notified about the acceptance or rejection via email.
