Adding Eligibility Criteria in Free-Form Text
The unstructured option for adding other eligibility criteria enables you to record eligibil­ity criteria in free-form text.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Add Eligibility Criteria in Free-Form Text
1.
On the navigation pane, click Eligibility Criteria.
The Interventional Trial Design – Eligibility Criteria page appears.If the Add Other Criterion button is not displayed, complete all fields on the Eligibility page first. To do so, see Abstracting Trial Eligibility Criteria.
2.
Click Add Other Criterion.
3.
The Add/Edit Eligibility Criteria Page appears.In the Eligibility Criterion Type field, indicate whether satisfying the criterion qualifies or disqualifies a subject from participation in the trial by selecting Inclusion or Exclusion respectively.
4.
Select the Unstructured option.
5.
In the Eligibility Criterion Display Order field, type the number that indicates the position that the current criterion will be displayed relative to other records in the Other Criteria table.
For example:
If the trial you are working on has been abstracted previously and displays four eligibility criteria records in the Other Criteria table, you can insert the criterion you are adding between the second and third records by typing 3.
6.
In the Description Text field, type a comprehensive description of the criterion.
7.
Click Save.
The new criterion record is displayed on the Eligibility Criteria page.
