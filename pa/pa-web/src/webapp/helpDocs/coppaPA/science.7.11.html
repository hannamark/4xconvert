Editing Other Eligibility Criteria
You can edit both the default mandatory criteria and other criteria that you or others may have added during abstraction.
How to Edit Other Eligibility Criteria
1.
On the navigation pane, click Eligibility Criteria.
The Eligibility Criteria page appears.
2.
To edit the required fields only, follow the instructions in Abstracting Interventional Trial Eligibility Criteria.
3.
To edit additional criteria, in the Edit column for the record you want to modify, click the Edit icon ().
The Add/Edit Eligibility Criteria page appears, displaying the criterion as submitted.
4.
Modify any or all of the information, and then click Save. See Adding Other Eligibility Criteria
