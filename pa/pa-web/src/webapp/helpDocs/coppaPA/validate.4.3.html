Selecting Trials that Require Validation
Submitted trials that are ready for validation are listed in search results tables along with trials in other stages of the processing life cycle (Processing Status). You can, however, list only those trials that require validation. You can achieve this by limiting your search to trials that have the processing status Submitted or you can sort an existing list of trials by the Action column.
How to Search for Trials that Require Validation
1.
Follow the instructions in Searching for Trials, using only the Processing Status criterion, Submitted.
To limit your search to complete trials, select Complete from the Search by Trial Category drop-down list on the Trial Search page.
Trials that require validation are listed in the search results table.
2.
In the Action column for the trial you want to validate, click Validate. Or, click the NCI Trial Identifier link.
The selected trial appears on the Trial Identification page with the navigation pane on the left side of the page. The system displays different submenus on the navigation pane depending on whether the trial is complete or abbreviated.
How to Sort Search Results by Validation Status
1.
In a search results table from any search, sort the trials by Action. To do so, click the Action column heading.
The trials are sorted by Action, in ascending order.
2.
Click the Action column heading again to reverse the sort order to display validation-ready trials at the top of the table.
