Reviewing IND/IDE Details
Ensure that details in existing records are accurate and that there are no duplicate records.
How to Review Trial Funding Details
1.
In the navigation pane, click Trial IND/IDE.
The Trial IND/IDE Information page appears and displays the following:
•
No records
•
One or more IND records
•
One or more IDE records
2.
Do one of the following to continue:
•
To add IDE or IND information if no records are displayed, click Add and follow the instructions provided in Abstracting Trial IND/IDE Information.
•
To modify an existing record, click the Edit icon () and follow the instructions provided in Abstracting Trial IND/IDE Information.
•
To delete a record, click the Delete icon () in the Delete column.
There may be more than one record in the table. Be sure you select the Edit or Delete icon in the same row as the record you want to modify.
3.
Proceed to Reviewing Trial Documents.
