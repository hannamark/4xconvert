Adding Interventions to a Trial
You can link interventions to a trial by searching for an intervention name from the current list of active records.
The terminology list is reloaded weekly.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Add Interventions
1.
On the navigation pane, click Interventions.
The Interventions page appears, with or without any intervention records displayed.
2.
Click Add
The Add intervention page appears.
3.
From the Intervention Type drop-down list, select the means of intervention for the trial. The following table lists valid intervention types.
Intervention Types
Drug
Device (including sham)
Biological/Vaccine
Procedure/Surgery
Radiation
Behavioral (for example, Psychotherapy, Lifestyle Counseling)
Genetic (including gene transfer, stem cell, and recombinant DNA)
Dietary Supplement (for example, vitamins, minerals)
Other
You can abstract details for drug, radiation, and surgical procedure interventions using Protocol Abstraction’s structured approach. To do so, see Generating Drug, Radiation, and Procedure/Surgery Descriptions.
4.
To search for the name of the intervention, click Look Up.
The Search Intervention window appears.
5.
In the Intervention Name field, type part or all of the intervention name.
Type as many letters of the intervention name as possible to reduce the number of search results. The system adds wildcards on both sides of the search string (the combination of letters you type) for you implicitly. You can type wildcard symbols (*) between characters of the string as necessary.
6.
To include synonyms in your search in addition to preferred names, select the Include Synonym check box.
7.
To limit your search to exact matches only, select the Exact Match Only check box.
8.
Click Search.
The system searches for the intervention from both preferred names and synonyms (if selected) in the database, and returns a list of results.
9.
Scroll through the list to locate the intervention, and click Select.
The selected intervention is displayed in the Intervention Name field.
10.
Optionally, in the Intervention Description field, type the name/description of the intervention. If you selected Drug, Radiation, or Procedure/Surgery from the Intervention Type list, you can generate a description instead. See Generating Drug, Radiation, and Procedure/Surgery Descriptions.
11.
If applicable, repeat the steps above to add additional records.
12.
Click Save.
The record is added to the trial and displayed on the Interventions page.
Although not displayed on the Interventions page, the system records the ClinicalTrials.gov code for the intervention.
