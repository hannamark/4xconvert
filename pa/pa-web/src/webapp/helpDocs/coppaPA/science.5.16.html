Adding Interventions to a Trial
You can link interventions to a trial by searching for an intervention name from the PDQ database.
How to Add Interventions
1.
On the navigation sidebar, click Intervention.
The Interventions page appears, with or without any intervention records displayed.
2.
Click Add
The Add intervention page appears.
3.
From the Intervention Type field, select the means of intervention for the trial. The following table lists valid intervention types.
Intervention Types
Drug
Device (including sham)
Biological/Vaccine
Procedure/Surgery
Radiation
Behavioral (for example, Psychotherapy, Lifestyle Counseling)
Genetic (including gene transfer, stem cell, and recombinant DNA)
Dietary Supplement (for example, vitamins, minerals)
Other
4.
If you selected Drug from the Intervention Type list, select the Lead Product check box for the record you are creating, or ensure that one drug record in the trial is designated the lead drug.
5.
To search the PDQ database for the name of the intervention, click Look Up.
The Search Intervention window appears.
6.
In the Intervention Name field, type part or all of the intervention name.
Type as many letters of the intervention name as possible to reduce the number of search results.
7.
Click Search.
The system searches for the intervention from both preferred names and synonyms in the database, and returns a list of results.
8.
Scroll through the list to locate the intervention.
9.
If the intervention appears in the list, click Select, and proceed to Step 12.
10.
If the intervention does not appear in the list, search for it in the Enterprise Vocabulary Services (EVS) repository in the NCI Bio Portal (http://bioportal.nci.nih.gov/ncbo/faces/pages/search.xhtml).
a.
If EVS lists the intervention, contact Protocol Abstraction Application Support to request that it be added to the list in Protocol Abstraction.
b.
If EVS does not list the intervention, contact EVS Application Support to request that it be added to the repository as a new concept.
11.
If applicable, repeat the steps above to link additional records.
12.
On the Add Intervention page click Save.
The record is added to the trial and displayed on the Interventions page .
