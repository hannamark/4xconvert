Setting Up Accrual Access
When you receive a CTRO request for access, you must associate the user with the appropriate trial/site. Thereafter you can grant or deny the user access without having to repeat the setup. See Managing Accrual Access for requirements for granting/denying Accrual site access.
Ηοω το Set Up a User for Accrual Access
1.
Search for the trial to which access has been requested. For detailed instructions, see Searching for Trials.
2.
Click the NCI Trial Identifier link.
3.
The Trial Identification page appears with the navigation pane displayed.On the navigation pane, in the Trial Overview section, click Manage Accrual Access.
4.
To add a user who has requested access, click Add.
5.
The Set Up Accrual Access page appears.From the User Name drop-down list, select the appropriate registered user.
The user’s contact information fields are populated automatically.
6.
From the Accessing Site drop-down list, select the appropriate participating site.
The Site Recruitment Status field is populated automatically.
7.
From the Access Status drop-down list, select either Active, to grant access, or Inactive, to deny access.
8.
Optionally, in the Request Details field, type the details of the request as recorded by the CTRO.
9.
Click Save, and then click OK on the pop-up window.
The record is added to the Manage Accrual Access page.
