Abstracting Trial Eligibility Criteria
The Trial Eligibility Criteria information provides a summary of the qualifications for subject enrollment in the study. It provides both “Inclusion” and “Exclusion” criteria.
Eligibility criteria include the set of pre-defined criteria (required) displayed on the default Eligibility Criteria page Abstracted trials may display other eligibility criteria listed in a table below the pre-defined criteria.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Abstract Eligibility Criteria
1.
On the navigation pane, click Eligibility Criteria.
The Interventional Trial Design – Eligibility Criteria page appears.
The Eligibility Criteria section displays the primary eligibility elements. The Other Criteria section may display other criteria as per ClinicalTrials.gov descriptions, and/or criteria that may have been added directly previously (see Adding Other Eligibility Criteria).
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. All fields are required.
Trial Description
Instruction
Accepts Healthy Volunteers
If the trial accepts healthy participants, select Yes. Otherwise, select No.
Gender
Indicate the gender criteria by selecting either Female, Male, or Both.
Minimum Age
Type the minimum participant age.
If there is no minimum age, type 0, and select Years from the Unit drop-down list.
Maximum Age
Type the maximum participant age.
If there is no maximum age, type 999, and select Years from the Unit drop-down list.
Unit
Select the age unit from the drop-down list.
3.
Click Save.
The trial is updated with the information you provided, and a new interface appears which enables you to add other criteria. See Adding Other Eligibility Criteria.
