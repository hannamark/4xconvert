Linking Contacts to a Trial
You must link at least one Participating Site to the trial before abstracting PI and contact information. You can add a contact by providing a person’s name (i.e., someone who is associated with the trial itself), or you can add a generic contact (i.e., someone who is associated with the site but not necessarily the trial) by providing a person’s title (functional role). You can not provide both types of contacts in the same record.
You will not be able to complete the abstraction if you do not provide a primary contact.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Add a Contact Name
1.
On the navigation pane, click Participating Sites.
2.
The Participating Sites Page appears.Click Add.
3.
The Participating Sites tab appears.Click the Contacts tab
The Participating Sites page displays the Contacts tab.
4.
If a primary investigator has been designated as the primary contact (on the Investigators tab), the investigator's name is displayed automatically on the Contact tab.Next to the First Name field, click Look Up and search for the contact person’s name by following the instructions in Looking Up Registered Persons.The person’s name you selected appears in the Name fields on the Contact tab.
5.
In the Phone Number field, type the contact’s primary telephone number (as 123-456-7890), including an extension if provided.
6.
In the Email Address field, type the contact’s primary email address.
7.
Click Save.
How to Add a Functional Role Contact
1.
Next to the Generic Contact field, click Look Up Generic Contact.
The Select Generic Contact window appears.
2.
In the Title field, type part or all of the contact’s role, and then click Search.
3.
If your search does not return any results, click Add Generic Contact.
a.
The Select Generic Contact window appears.Type the appropriate information in the Title, Email, and Phone fields, and then click Save.
4.
In the Action column, click Select.
The Participating Sites page displays the contact information you selected.
5.
Click Save.
