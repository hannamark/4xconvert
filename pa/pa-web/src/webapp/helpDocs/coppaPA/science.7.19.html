Editing Intervention Records
You can edit the type of intervention and the intervention description only for any record. If the intervention type is a drug, you can also change the lead drug designation.
How to Edit Intervention Records
1.
On the navigation pane, click Intervention.
The Interventions page appears.
2.
In the Edit column for a given record, click the Edit icon ().
The Edit Intervention page appears.
3.
To change the intervention type, select the appropriate intervention from the Intervention Type drop-down list.
4.
To change the Lead Product designation for drug interventions, select or clear the Lead Product check box.
5.
In the Intervention Description field, make any changes as necessary.
6.
Click Save.
