Abstracting Interventional Trial Eligibility Criteria
The Trial Eligibility Criteria information provides a summary of the qualifications for subject enrollment in the study. It provides both “Inclusion” and “Exclusion” criteria.
Eligibility criteria include the set of pre-defined criteria (required) displayed on the default Eligibility Criteria page and, in some trials, other eligibility criteria listed in a table below the pre-defined criteria.
How to Abstract Eligibility Criteria
1.
On the navigation sidebar, click Eligibility Criteria.
The Interventional Trial Design – Eligibility Criteria page appears.
2.
If the trial accepts healthy participants, in the Accepts Healthy Volunteers field, select Yes. Otherwise, select No.
3.
In the Gender field, indicate the gender criteria by selecting either Female, Male, or Both.
4.
In the Minimum Age field, type the minimum participant age, and then select the age unit from the Unit drop-down list. If there is no minimum age, type 0.
5.
In the Maximum Age field, type the maximum participant age, and then select the age unit from the Unit drop-down list. If there is no maximum age, type 999, and select Years from the Unit drop-down list.
6.
Click Save.
The trial is updated with the information you provided, and a new interface appears which enables you to add other criteria. See Adding Eligibility Criteria
