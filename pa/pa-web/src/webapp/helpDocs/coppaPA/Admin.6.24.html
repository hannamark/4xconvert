Adding Trial-Related Documents
During abstraction you can add missing trial-related documents or replace the ones that currently exist.
How to Add Trial Related Documents
1.
On the navigation pane, click Trial Related Documents.
The Trial Related Documents page appears.
2.
Click Add.
The Add/Edit New Document page appears.
3.
In the Document Type field, select the type of trial document you want to add to the abstract from the drop-down list. The following table lists valid document types.
Trial Related Document Type
Protocol
IRB approval
Informed Consent
Participating sites
Other
Amendment protocol
Change memo
4.
Click Browse, navigate to the document you want to upload in the File Upload window, and then click Open.
Depending on your operating system, the “File Upload” window and the “Open” button may be named something else.
The Trial Related documents page displays your new document in the table.
