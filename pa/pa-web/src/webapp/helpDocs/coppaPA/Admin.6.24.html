Abstracting Trial-Related Documents
Original trial-related documents and amended trial documents have the following constraints:
•
Only one document of each following types is permitted:
•
Protocol document
•
IRB approval
•
Participating sites
•
Informed consent
•
Change memo
•
Amended protocol
•
TSR
•
The following documents are not required if the information they contain is included in the protocol document:
•
Participating sites
•
Informed consent
•
A trial can have multiple “other” documents
•
The following new documents must replaced existing ones (that is, there can be no duplicates):
•
Protocol documents
•
Change memos
•
Amended protocols
•
IRB approval documents
Each trial submitted must include the following types of documents:
•
For non-proprietary trials only: Protocol Document (complete protocol)
•
For proprietary trials only: A summary of the protocol in lieu of a Protocol Document. A summary may not be necessary if the proprietary trial includes a proprietary template. The NCT document number is mandatory.
•
IRB approval
•
Informed Consent (if not included in the protocol document)
Note:
If the informed consent information is not included as part of the proto­col document, you can upload Informed Consent document.
•
Participating sites
If the Participating Sites information is not included as part of the protocol document, you can upload it.
•
participating sites
•
investigators
•
primary contact information
•
collaborators
If collaborators are not listed in the protocol document, they may be listed in the Participating Sites document.
Every trial amendment submitted must contain one of each of the following types of documents in addition to the other documents required for the original submission:
•
Amended protocol
•
Change memo – Summary of changes to the protocol
You are required to supply your documents as Microsoft Word, Adobe PDF, Microsoft Excel, and WordPerfect files.
