Validating Abstractions
As soon as you click Abstraction Validation on the navigation pane, the system scans the abstracted information for errors and omissions automatically. You must correct all errors before completing the abstraction.
The instructions below apply to trials with the current trial processing status Accepted. Refer to Trial Processing Statuses.
How to Validate Abstractions
1.
On the navigation pane, click Abstraction Validation.
The Abstraction Validation page appears. If the system does not find any errors, it displays â€œAbstraction is valid. If the abstraction contains errors or omissions, they are displayed on this page. Instructions for correcting each error/omission are provided in the Comment column.
2.
If the Abstraction Validation page contains error/omission records, correct them according to the instructions in the Comment column and specific instructions included throughout the online help, and then proceed to the next step.
3.
Optionally, if the Abstraction Validation page does not contain error/omission records, click View TSR or View XML to review the Trial Summary Report (TSR) or XML file.
When the abstraction has been verified, the trial processing status is updated to Abstraction Verified.
