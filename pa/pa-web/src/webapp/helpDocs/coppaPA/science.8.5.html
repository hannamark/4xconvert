Adding Outcome Measures
You can add multiple primary and secondary outcome measures to a trial.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Add Outcome Measures
1.
On the navigation pane, click Outcome Measures.
The Outcome Measures page appears, with or without any outcome measure records displayed.
2.
Click Add.
The Add/Edit Outcome Measure page appears.
3.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. All fields are required.
Trial Description
Instruction
Primary
•
To add a primary outcome measure, select Yes.
- or -
•
To add a secondary outcome measure, select No.
Title
Type the title of the outcome measure.The maximum number of characters is 254.
Description
Type a description of the outcome measure. For example, Progression-free survival or Frequency and severity of adverse events. The maximum number of characters is 600.
Time Frame
Type the duration of the outcome measure. The maximum number of characters is 254.
Safety Issue
If the outcome measure presented a safety issue, select Yes. Otherwise, select No.
4.
Click Save.
