Deleting Participating Sites From a Trial
If a participating site currently linked to the trial you are abstracting is incorrect, you can delete the site.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Delete a Participating Site From a Trial
1.
On the navigation pane, click Participating Sites.
The Participating Sites Page appears.
2.
To disassociate the participating site from the trial, click the Delete icon () in the Delete column, and then confirm the action in the pop-up window.
There may be more than one record in the table. Be sure you select the Delete icon in the same row as the record you want to delete.
