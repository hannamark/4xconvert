Editing Participating Site Information
If a participating site currently linked to the trial you are abstracting is incorrect, you can edit the site information.
How to Edit Participating Sites Information
1.
On the navigation pane, click Participating Sites.
The Participating Sites Page appears.
To make changes to the record, click the Edit icon () in the Edit column.
2.
Modify the information by following the instructions for the following tasks:
•
Linking Organizations to a Trial
•
Linking Investigators to a Trial
•
Linking Contacts
