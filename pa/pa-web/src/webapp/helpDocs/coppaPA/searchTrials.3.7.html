Accessing Trials in the Inbox
The Inbox lists trials in the CTRP that have been updated by a trial submitter and require re-abstraction. You can search for specific trials by their NCI Trial Identifiers. Trials in the Inbox have reached the QC Completed Date milestone.
You can check all trials in the Inbox by typing in letter N in the NCI Trial Identifier field.
The Inbox also lists administrative access requests from NCI CTRP Registration Site users. These are listed by default in the Pending Admin User section; you do not have to search for them. However, when you search for a trial, only the list of search results will be displayed.
To display the list of Pending Admin User requests following a search for updated trials, click the Inbox menu on the navigation pane again.
You can review or abstract any updated trial record listed in the Inbox. Once listed, you can sort the updated trial records by clicking the column headings labeled with bold text.
Remember to remove the trial from the Inbox once you have completed the abstraction. See Removing Trials From the Inbox.
How to Search for Trials in the Inbox
1.
On the top section of the navigation pane, click Inbox.
The Search Trial page appears. Requests for Site Administration, if any, are listed under Pending Admin User by default.
2.
Type one or more characters in the NCI Trial Identifier field, and then click Submit.
All updated trials matching your criterion are listed in the Search Results section.
3.
To abstract a trial, in the Action column for the trial record you want to abstract, click Abstract.
In order to initiate abstraction of a trial in the Inbox, you must lock the trial. See Checking In and Checking Out Trials.
