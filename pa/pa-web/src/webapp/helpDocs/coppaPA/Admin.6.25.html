Abstracting Trial-Related Documents
Original trial-related documents and amended trial documents have the following constraints:
•
Only one document of each following types is permitted:
•
Protocol document
•
IRB approval
•
Participating sites
•
Informed consent
•
Change memo
•
TSR
•
Protocol highlight
•
A trial can have multiple “other” documents
•
The following new documents must replaced existing ones (that is, there can be no duplicates):
•
Protocol documents
•
Change memos
•
Informed consent
•
IRB approval documents
•
Protocol highlight
•
TSR
Each trial you submit must include the following types of documents:
•
For original submission of non-proprietary trials only: Protocol Document (complete protocol) and IRB approval. Informed consent and participating site documents are optional and can be uploaded if relevant information is not included in the protocol document.
•
For proprietary trials only: Protocol template.
•
Participating sites
If the Participating Sites information is not included as part of the protocol document, you can upload it.
•
participating sites
•
investigators
•
primary contact information
•
collaborators
If collaborators are not listed in the protocol document, they may be listed in the Participating Sites document.
Every trial amendment you submit must contain one of each of the following types of documents:
•
IRB approval document
•
Informed consent
•
Protocol highlight change
•
Participating site documents (if appropriate)
•
Amended protocol
•
Change memo – Summary of changes to the protocol
You are required to supply your documents as Microsoft Word, Adobe PDF, Microsoft Excel, and WordPerfect files.
