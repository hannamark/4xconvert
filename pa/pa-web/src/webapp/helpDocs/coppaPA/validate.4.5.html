Reviewing Trial Overview Details
You can edit or add most trial overview information except the identifiers that the system generates on the Trial Identification page.
Trial Overview data for trials comprises the following information:
•
Trial Identification
•
Trial History – See Abstracting Trial History Information.
•
Trial Milestones – See Abstracting Trial Milestones.
•
On-hold Information – See Abstracting On-hold Information.
•
View TSR – See Accessing Trial Summary Reports.
Use the navigation pane to navigate easily to the specific category of data you would like to review.
The trial summary remains displayed at the top of each page as you navigate through the Trial Overview section This serves as a visual reminder of the trial you are abstracting or editing.
