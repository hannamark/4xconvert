Deleting Trial-Related Documents
Protocol Abstraction enables you to delete some current trial-related documents. Although you can not delete protocol or IRB approval documents, you can replace them. See Editing and Replacing Documents.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Delete Trial-Related Documents
1.
On the navigation pane, click Trial Related Documents.
The Trial Related Documents page appears.
2.
Click the Delete icon.
The Trial Related Document page provides a text box in which you must state the reason for deleting the document, for example, “Duplicate document”.
3.
Click Done.
