Searching for Trials
You can search for—and abstract or edit—any registered trial that has been submitted successfully to the Clinical Trials Reporting Program (CTRP). Search criteria are available to locate a particular trial to work on, or you can list all registered trials that are available for validation and abstraction. You also can limit your search to the trials you have checked out previously.
Additionally, you can access and manage any registered trial that has been updated. See Processing Updated Trials and Requests for Admin Rights.

1.
If the Trial Search page is not already displayed, on the Protocol Abstraction navigation pane on the left side of the page, click Trial Search.
The Trial Search page appears.
You can create more space on your monitor for the search results list by hiding the search fields. To do so, select the Hide Search Fields check box at the top left side of the page.
