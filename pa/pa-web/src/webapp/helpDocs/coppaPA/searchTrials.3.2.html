Searching for Trials
You can search for—and abstract or edit—any registered trial that has been submitted successfully to the Clinical Trials Reporting Program (CTRP). Search criteria are available for each to locate a particular trial to work on, or you can list all registered trials that are available for abstraction and validation. You also can limit your search to the trials you have checked out previously.
Additionally, you can access and manage any registered trial that has been updated. See Processing Updated Trials in the Inbox
How to Search for Trials
1.
If the Trial Search page is not already displayed, on the Protocol Abstraction navigation pane on the left side of the page, click Trial Search.
The Trial Search page appears.
You can create more space on your monitor for the search results list by hiding the search fields. To do so, select the Hide Search Fields check box at the top left side of the page.
2.
Do one of the following to search for and select available trials in the system:
•
To list all trials, leave all fields blank, and then click Search.
- or -
•
To search for a particular trial or set of trials, continue with the next steps.
3.
Type or select any of the search criteria in the following table as appropriate.
Search Criteria
Description
NCI Trial Identifier
Type all or part of the identifier that NCI assigned to the trial when it was registered.
Lead Organization Trial ID
Type all or part of the numeric or alphanumeric identifier assigned to the study by the lead organization. Inter-Group trials use the lead Groups trial number.
Principal Investigator
Select the principal investigator from the drop-down list.
To narrow the list of investigators displayed, click the Principal Investigator field and type the initial letter(s) of the last name. You can also use the Up and Down arrow keys on your keyboard to scroll through the list.
