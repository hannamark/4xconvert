Editing and Replacing Documents
Protocol Abstraction enables you to edit current trial related documents by downloading them, editing them, and replacing them.
To delete, rather than edit, documents other than protocol and IRB approval documents, see Deleting Trial-Related Documents.
How to Edit and Replace Trial Related Documents
1.
On the navigation pane, click Trial Related Documents.
The Trial Related Documents page appears.
2.
To replace a document, click the Edit icon () in the Edit column and upload a replacement document as per the instructions in Step 4 below. The replacement document must be of the same type as the original.
The document is replaced and appears in the table. Continue with the steps below only if you want to modify an existing file.
There may be more than one record in the table. Be sure you select the Delete icon in the same row as the record you want to delete.
3.
To edit a document, click its file name link, save the file to a location on your local drive, and modify the file as necessary. You can save the modified file with the same name (that is, write over the original), or save it with a new file name.
4.
On the Trial Related Documents page, click the Edit icon () in the row of the document you downloaded.
The Add/Edit New Document page appears. Note that you are not able to change the document type.
5.
Click Browse, navigate to the document you just edited, and then click Open.
Depending on your operating system, the “File Upload” window and the “Open” button may be named something else.
The Trial Related documents page displays your modified document in the table.
