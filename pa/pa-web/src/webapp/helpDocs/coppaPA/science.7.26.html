Abstracting Arm Information
Complete this section for non-proprietary trials.
In interventional studies, an arm is a group of study subjects associated with a given treatment assignment. When adding arms to a trial, you assign to them the interventions currently abstracted for the trial. You must therefore complete intervention abstraction before abstracting arms or updating arms after interventions have been recorded.
The following rules apply to each trial:
•
There must be at least one arm that implies that the entire study subject population receives the same intervention.
•
The number of arms added to the trial must coincide with the number indicated in the trial’s “Design Details.” See Abstracting Trial Design.
•
Every arm must have at least one assigned intervention (except for the arm type “no intervention”).
•
Every intervention must be assigned to at least one arm.
•
There must be only one arm with the same combination of description and associated intervention.
Trials can have multiple arm records. Each record displays the following arm information:
Column Name
Definition
Label
Short name used to identify the arm.
Type
Function of the arm. Valid types are as follows:
•
Experimental
•
Active Comparator
•
Placebo Comparator
•
Sham Comparator
•
No intervention
•
Other
Description
Brief description of the arm. This element is mandatory for at least one arm in the trial.
Assigned Interventions
Intervention assigned to a given arm.
