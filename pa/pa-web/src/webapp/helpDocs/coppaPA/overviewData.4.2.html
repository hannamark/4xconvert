About Trial Overview Data
You can edit or add most trial overview information except the identifiers that the system generates in the Trial Identification page.
Trial Overview data for trials comprises the following information:
•
Trial Identification
•
Trial History
•
Trial Milestones
•
On-hold Information
You can use the navigation pane to navigate easily to the specific category of data you would like to add or edit.
The trial summary remains displayed at the top of each page as you navigate through the Trial Overview section This serves as a visual reminder of the trial you are abstracting or editing.
