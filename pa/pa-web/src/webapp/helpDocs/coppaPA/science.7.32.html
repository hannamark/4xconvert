Editing Sub-group Records
You can edit sub-group records as necessary.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Edit Sub-group Records
1.
On the navigation pane, click Sub-group.
2.
The Sub-groups page appears.In the Edit column for a given record, click the Edit icon ().
3.
The Add/Edit Sub-group page appears.Modify the sub-group details according to instructions in Adding Sub-groups to a Trial.
4.
Click Save.
