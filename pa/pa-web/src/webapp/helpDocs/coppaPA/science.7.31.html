Adding Sub-groups to a Trial
You can add one or more sub-groups to a trial according to the trialâ€™s design details. See Abstracting Trial Design.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Add Sub-groups
1.
On the navigation pane, click Sub-groups.
The Sub-groups page appears, with or without any sub-group records displayed.
2.
Click Add
3.
The Add/Edit Sub-group page appears.In the Code field, type a unique sub-group identifier.
4.
In the Description field, type a brief description of the sub-group.
5.
Click Save.
6.
Repeat the steps above until all sub-groups have been added.
