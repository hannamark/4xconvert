Abstracting Interventions
Complete this section for both non-proprietary and proprietary trials.
Protocol Abstraction enables you to search for and select interventions from a list of active intervention terms extracted from PDQ and link them to a trial. PDQ is an online database developed and maintained by the National Cancer Institute. You can find more specific information about PDQ at http://cancer.gov/cancer_information/pdq/.
Additionally, Protocol Abstraction provides a structure for abstracting drug, radiation, and surgical procedure details.
Some intervention terms become obsolete over time. When you validate the trial, the system displays an error message if the abstraction contains an obsolete term. If this occurs, look up the intervention name and replace the obsolete term with the new one.
Trials can have multiple intervention records. Each record displays the following information about the disease or conditions:
Column Name
Definition
Name
Standard name used to refer to each intervention. For drugs, this is the generic name. For investigational new drugs that do not yet have a generic name, this may be the chemical name, company code, or serial number.
Other Names
Synonym or other name.
Description
For drug interventions, a summary of the drug’s chemical nature and (potential) mechanism of action, and/or details such as dosage form, dosage, frequency and duration.
For other interventions, a summary of the procedure or device that provides key details that distinguish it from similar interventions.
Type
Mode of intervention, for example, drug or device.
Lead?
For drug interventions, “Yes” in the “Lead” column indicates that the drug is the lead drug on trial. Each trial can have one lead drug only.
Additionally, you can add, edit, and delete records.
When abstracting intervention information, ensure that each trial that includes one or more drug interventions has one lead drug.
