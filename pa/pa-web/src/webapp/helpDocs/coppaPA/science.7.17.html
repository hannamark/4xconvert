Abstracting Interventions
Protocol Abstraction enables you to search for and select interventions from the PDQ (Physician Data Query) database and link them to a trial. PDQ is an online database developed and maintained by the National Cancer Institute. You can find more specific information about PDQ at http://cancer.gov/cancer_information/pdq/.
Some intervention terms in PDQ become obsolete over time. When you validate the trial, the system displays an error message if the abstraction contains an obsolete term. If this occurs, look up the intervention name and replace the obsolete term with the new one.
Trials can have multiple intervention records. Each record displays the following information about the disease or conditions:
Column Name
Definition
Name
Standard name used to refer to each intervention. For drugs, this is the generic name. For investigational new drugs that do not yet have a generic name, this may be the chemical name, company code, or serial number.
Other Name
Synonym or other name.
Description
For drug interventions, a summary of the drug’s chemical nature and (potential) mechanism of action, and/or details such as dosage form, dosage, frequency and duration.
For other interventions, a summary of the procedure or device that provides key details that distinguish it from similar interventions.
Type
Mode of intervention, for example, drug or device.
Lead?
“Yes” in the “Lead” column indicates the primary drug. Interventions other than drugs do not require a lead.
Additionally, you can add, edit, and unlink records.
