Abstracting General Trial Details
Before you begin abstracting any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
General trial details include the following main categories of data:
•
ClinicalTrials.gov XML Required indicator
•
Trial title
•
Trial description
•
Lead organization/Principal investigator
•
Sponsor Responsible party (if applicable)
•
Central contact
Unless otherwise noted, you are required to complete all information provided on the General Trial Details page. Most elements are provided via the CTRP Registration Site.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Abstract General Trial Details
1.
On the navigation pane, click General Trial Details.
The General Trials Details page appears.
2.
Select the appropriate option to indicate whether an XML document is required to register the trial with ClinicatTrials.gov.
3.
Trial submitters are required to provide a Lead Organization Trial Identifier. If the number is not correct, in the Lead Organization Trial Identifier field, type the unique protocol ID assigned to the protocol by the lead (or sponsoring) organization.
4.
Optionally, in the NCT Number, CTEP, and DCP Indentifier fields, type the trial identifier assigned by ClincalTrials.gov.
