Validating Trial Details
The final step before accepting or rejecting a trial is to validate the trial details.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Validate Trial Details
1.
In the navigation pane, click Trial Validation.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. All items marked with an asterisk (*) are required fields.
Field Label
Instructions
Lead Organization Trial ID*
Ensure that the ID matches the one provided in the protocol. Make changes as necessary.
See Abstracting General Trial Details.
NCT Number
See Abstracting General Trial Details.
Official Title*
Type or correct the trial title as necessary. You can use a maximum of 4000 characters.
See Abstracting General Trial Details.
Trial Phase*
See Abstracting Trial Design.
Primary Purpose*
See Abstracting Trial Design.
Primary Purpose Other Text (available if Primary Purpose = “Other”)
See Abstracting Trial Design.
Lead Organization*
See Abstracting General Trial Details.
Summary 4 Funding Category
See Abstracting NCI-Specific Information
Summary 4 Funding Source
See Abstracting NCI-Specific Information
Program Code
See Abstracting NCI-Specific Information
3.
Do one of the following to continue:
•
To save the trial information so that you can continue to validate the trial at a later time, click Save.
- or -
•
To accept or reject the trial, follow the instructions in Accepting and Rejecting Trials.
