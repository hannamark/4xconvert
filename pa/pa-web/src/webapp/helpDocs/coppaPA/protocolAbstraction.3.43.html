Editing Trials
Only a documents specialist can edit trials. Only trials that have not been accepted or rejected can be edited.
To edit a trial, follow these steps:
1.
Retrieve the trial you want to edit.
2.
You can open a trial to edit in two ways:
•
Click the Edit link.
•
Click the NCI identifier to view trial details then click Edit.
The Edit a Trial page appears.
3.
Edit the fields, described in the following table, as needed. Asterisks indicate required fields.:

Trial Status Date
Enter the date the trial status was assigned to the trial, using the format mm/dd/yyyy, or select the date from the calendar to automatically populate the field.
Lead Organization
Specify the lead organization. You must select one of the following choices:
•
From the Lead Organization drop-down menu, select the organization primarily responsible for administering the trial.
•
To add a new lead organization, click Add New and enter the name of the organization.
Principal Investigator
Specify a principal investigator. You must select one of the following choices:
•
In the Principal Investigator drop-down menu which lists those already added to the Portal, select the person responsible for all aspects of the conduct of the study. .
•
To add a new principal investigator, click Add New and enter the full first name, the middle initial, and the full last name of the person responsible for the conduct of a clinical trial.
NIH Grant Information
Select from the drop-down lists or enter the appropriate information. A grant number is a concatenation of a number of elements. A decomposition of a grant number follows.
Grant number example:
1R01CA009999-01A1S2
Type: A single-digit code at position 1
Funding Mechanism: A three-character code, position 2-4
Institute Code: A two-character code, position 5-6
Serial Number: A 5 or 6 digit number, position 7 + till dash
Year: A two-digit number, position 1- 2 after dash
Suffix: String, position 3 + last after the dash.
Protocol Document
The three document types required by the system are noted in this section of the page.
Select the document type, then click Browse and locate and upload a document. You can add documents that are in Adobe PDF, Microsoft Word, Excel or WordPerfect format.
Comments
Add any comments or information not covered by the other fields in the page.
4.
Click one of the following buttons to complete your edit:
•
Save Edit implies saving changes to the record.
•
On Hold postpones the decision on submission validation due to expected delay (need to contact the submitter) or unknown delay (for getting submitter response).
•
Accept means that the submission is valid and the record becomes available for abstraction.
•
Reject means that the submission is invalid (or a duplication) and is excluded from abstraction.
