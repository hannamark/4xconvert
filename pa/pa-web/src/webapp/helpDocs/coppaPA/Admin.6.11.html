Abstracting Human Subject Safety Information
The information you are required to provide depends on the Board approval status and other indicators, as noted in the instructions that follow.
Note:
You will need the IRB approval document for a single site to abstract this category of information.
Before you begin to abstract human subject safety information, ensure that the participating sites information has been completed.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
Note:
The only field that you are required to complete in this section is the Board Approval Status. Other fields are either optional or non-editable, which are indicated by gray fields).
How to Abstract Human Subject Safety Information
1.
On the navigation pane, click Human Subject Safety.
2.
The Human Subject Safety Information section appears.In the Board Approval Status drop-down list, select the current state of the trial with respect to Investigational Review Board (IRB) approval. The Board Approval Status must correlate with the current Trial Status as follows:
•
If the current Trial Status is In Review, the Board Approval Status must be Submitted, pending.
•
If the current Trial Status is Disapproved, the Board Approval Status must be Submitted, Denied.
For other rules regarding Board Approval Status, see Trial Completion Criteria.
3.
The following table lists valid approval statuses.
Status
Definition
Submitted, Approved
IRB approval has been requested and obtained
Submitted, Exempt
IRB has granted an exemption in response to the approval request
Submission Not Required
Study does not require human subjects review
Submitted, Pending
Study in pending IRB review
Submitted, Denied
IRB has been denied approval for the study
4.
If you selected anything other than Submission Not Required, continue with the next steps. Otherwise you are finished with human subject safety abstraction.
5.
If the trial has been submitted and approved, type the number assigned to the protocol by the human subjects review board in the Board Approval Number field. If the number does not appear in the IRB approval document, type the date of submission using the mm/dd/yyyy format.
6.
Click Look Up to search for and select the Board name. If the name is not listed, register it with the CTRO. See Working With Organization Records.
When you select the Board name, the contact information is displayed automatically in the fields provided.
Note:
If you registered a new organization with the CTRO in the previous step, the system will notify you when the record is added so that you may continue your abstraction.
7.
If applicable, complete any of the contact fields that are missing information.
8.
Optionally, type the name of the affiliated organization in the Board Affiliation field.
