Overview of Protocol Abstraction
The NCI CTRP Protocol Abstraction web application enables authorized users to query, access, and validate submitted protocols; and to abstract accepted protocols through a common user interface that ensures standardization of data across clinical trials.
Protocol Abstraction supports Internet Explorer 7 and FireFox 3 (3.0.19) browsers.
Protocol Abstraction version 3.3 has the following new features and improvements:
•
An improved trial check-in and check-out user interface simplifies the check in/check out process.
•
New trial categorization. In previous versions, trial categorization was based on protocol document availability (i.e. whether a protocol was proprietary or non-proprietary). In this release, the trial categorization is based on the Summary 4 Funding Sponsor Type as follows:
•
Abbreviated trials. Trials with Summary 4 Funding Sponsor Type “Industrial”. In earlier releases, these trials were referred to as “Proprietary”.
•
Complete trials. Trials with Summary 4 Funding Sponsor Type “National”, “Externally Peer-Reviewed”, and “Institutional”. In earlier releases, these trials were referred to as “Non-Proprietary”.
•
You can indicate whether a trial IND/IDE is exempt from reporting to ClinicalTrials.gov. FDA-regulated intervention rules do not apply to trials with IND/IDE exemptions.
•
The valid values available for Trial Phase are the same as the ClinicalTrials.gov values.
•
You can now indicate whether a trial is a Pilot.
•
The phase additional qualifier value “Pilot” is now linked to the phase value “N/A”.
•
The valid values available for Primary Purpose are the same as the ClinicalTrials.gov values.
•
You can now describe the Primary Purpose in text form if you select the Purpose value “Other”.
•
Trial Summary Reports (TSRs) for registered trials, including trials registered prior to this release, are now available as Word and HTML documents instead of PDFs.
•
Improved Inbox processing streamlines Admin request processing.
•
When rejecting an Admin request, you can include a reason for rejection.
•
Summary 4 Funding Sponsor Type and Summary 4 Funding Sponsor Organization values are mandatory.
•
You can now change the Trial Status “In Review” to “Active”, and from “Closed to Accrual” to “Completed” when updating and amending trials.
•
New trial search criteria now include the following:
•
Identifier Type “Other”
•
Category (Complete or Abbreviated)
•
Submission Type “Update”
