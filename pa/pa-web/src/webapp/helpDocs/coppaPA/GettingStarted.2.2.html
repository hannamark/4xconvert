Overview of Protocol Abstraction
The NCI CTRP Protocol Abstraction web application enables authorized users to query, access, and validate submitted protocols; and to abstract accepted protocols through a common user interface that ensures standardization of data across clinical trials.
Protocol Abstraction supports Internet Explorer 7 and FireFox 3 (3.0.19) browsers.
Protocol Abstraction version 3.2 has the following new features:
•
Greater security
•
Ability to process requests for site administration status from NCI CTRP Registration Site users
•
Ability to assign trial ownership
•
Ability to reject trials after they have been accepted by CTRO staff (late rejection)
•
Ability to validate and abstract non-proprietary trials based on their requirements for submission of XML documents for registration with ClinicalTrials.gov
•
Ability to abstract OTHER trial identifiers
•
Ability to abstract local trial identifiers for participating sites (non-proprietary trial)
•
Ability to view and process selected updates for proprietary trials
