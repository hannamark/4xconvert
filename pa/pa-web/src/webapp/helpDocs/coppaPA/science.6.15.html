Adding Diseases/Conditions to a Trial
In this release of Protocol Abstraction, you can link diseases and conditions to a trial by searching for a disease name or condition name from the PDQ database.
How to Add Diseases/Conditions
1.
On the navigation pane, click Disease/Condition.
The Trial Disease/Condition page appears, with or without any disease or condition records displayed.
2.
Click Add
The Add Disease/Condition page appears.
3.
To search the PDQ database for the name of the disease or condition being studied, click Look Up.
The Search Disease Window appears.
4.
In the Disease Name field, type part or all of the disease/condition being studied.
Type as many letters of the disease or condition as possible to reduce the number of search results. You can use an asterisk (*) as a wild-card character to expand your search. If you do not use an asterisk, the search returns an exact match with whatever you type in this field.
5.
Click Search.
The system searches for the disease/condition from both preferred names and synonyms in the database, and returns a list of results.
Note:
The system will not return any results if you have typed too few characÂ­ters in the Disease Name field or have typed a disease/condition that has numerous sub-concepts.
6.
Scroll through the list (if needed) to locate the disease/condition being studied.
Some records in the results list may not be suitable for selection because they are not high enough in the concept hierarchy.
7.
If the disease/condition being studied appears in the list, click Select, and proceed to Step 9.
8.
If the disease/condition being studied does not appear in the list, search for it in the Enterprise Vocabulary Services (EVS) repository in the NCI Bio Portal (http://bioportal.nci.nih.gov/ncbo/faces/pages/search.xhtml).
a.
If EVS lists the disease/condition, contact Protocol Abstraction Application Support to request that it be added to the list in Protocol Abstraction.
b.
If EVS does not list the disease/condition, contact EVS Application Support to request that it be added to the repository as a new concept.
9.
If applicable, repeat the steps above to link additional records.
10.
On the Add Disease/Condition page for the appropriate record, select the Lead check box.
You must designate one lead disease/condition in order to be able to complete the abstraction.
11.
Click Save.
The record is added to the trial and displayed on the Disease/Condition page.
