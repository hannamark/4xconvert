Building Eligibility Criteria
The structured option enables you to build eligibility criteria based on measurable values and Cancer Data Standards Registry and Repository (caDSR)-defined eligibility criteria.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Build Eligibility Criteria
1.
On the navigation pane, click Eligibility Criteria.
The Interventional Trial Design – Eligibility Criteria page appears.
If the Add Other Criteria button is not displayed, complete all fields on the Eligibility page first. To do so, see Abstracting Trial Eligibility Criteria.
2.
Click Add Other Criteria.
3.
The Add/Edit Eligibility Criteria Page appears.Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. All fields are optional.
Trial Description
Instruction
Eligibility Criterion Type
Indicate whether satisfying the criterion qualifies or disqualifies a subject from participation in the trial by selecting Inclusion or Exclusion respectively.
Structured or Unstructured
Select the Structured option.
Eligibility Criterion Display Order
Type the number that indicates the position that the current criterion will be displayed relative to other records in the Other Criteria table.
For example:
If the trial you are working on has been abstracted previously and displays four eligibility criteria records in the Other Criteria table, you can insert the criterion you are adding between the second and third records by typing 3.
4.
Next to the Eligibility Criteria Name field, click Look Up.
The caDSR Lookup window appears
5.
In the Data Element Name field, type one or more characters of the criterion name.
6.
In the Category Code field, select the code for the criterion’s associated category.
To display all eligibility criteria for a given code, leave the Data Element Name field blank, and then click Search.
7.
Click Search.
The caDSR window displays the result(s) of your search.
8.
If the search did not return appropriate results, continue with the next steps. Otherwise, skip to Step 9.
a.
To request that caDSR create a new common data element (CDE), click CDE Create Request.
The caDSR window displays a request form. The Email field is pre-populated with the Application Support email address.
b.
In the Sender’s Email field, type your email address or the address of someone that the NCICB Applications Support staff can contact regarding your request.
Application Support staff email a message to you that contains the status of your request.
c.
In the Message field, type the text of the requested CDE, using the template provided.
d.
Click Send Email, then click the Close Window icon on the upper right corner of the caDSR Lookup window.
9.
In the search results list, locate the preferred criterion name, and then click Select.
The system enters the category you chose in the Category Code field, and displays the criterion name you selected.
10.
If the Eligibility Criterion Name falls into the Labwork category code, from the Lab Test Name drop-down list, select the appropriate lab test.
11.
In the Operator field, select the notation that establishes the upper or lower limits of the criterion.
For example:
•
= (equals)
•
> (greater than)
•
>= (greater than or equal to)
•
< (less than)
•
=< (less than or equal to)
•
In
e.
From the Value drop-down list, select the item used for comparison/evaluation.
For example, if you selected the criterion name, Performance Status Assessment, the value may be “Requires Considerable Assistance...”
12.
To use a caDSR unit of measure, next to the Unit field, click Look Up.
The Unit of Measure window appears.
13.
Type your search criteria in the field(s) provided, and then click Search.
You must type at least one character in any of the search fields.
The Unit of Measure window displays the result(s) of your search.
14.
In the search results list, locate the preferred unit of measure, and then click Select.
The system displays the unit of measure you selected.
15.
Click Generate Description Text.
The system displays the criteria you selected in text form in the Description text box.
You can modify the generated description as necessary.
16.
Click Save.
17.
In the pop-up message, click OK to save your record.
The new criterion record is displayed on the Eligibility Criteria page.
