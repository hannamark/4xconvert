Abstracting Trial History Information
The Trial History page displays information about all existing trial submissions except those that have been rejected. This page provides the means to validate and abstract amendment-specific information, and to review all submission-specific documents. It also displays trial update history, and access to trials that have been updated but not verified.
Trial submitters can update trials any time after the CTRO has accepted their trials. As a result, the subset of updated trials that require re-abstraction, and those that have been updated in the past, are listed on the Trial History Information Updates tab.
The Inbox also lists administrative access requests from NCI CTRP Registration Site users. See Processing Requests for Admin Rights.
Trial updates do not affect any elements that are reflected in the protocol document. Changes to the protocol document that require IRB approval are submitted via trial amendments.
Before you begin abstracting any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data. For instructions, see Checking In and Checking Out Trials.
When an abstractor rejects an amendment, the system deletes the amendment information and sends an email message to the submitter that contains the reason for rejection.
How to Abstract Trial Submission History Information
1.
In the navigation pane, click Trial History.
The Trial History Information page displays the Submission tab by default. It provides read-only access to the original and amendment-specific documents, and other historical information.
2.
To review the original trial-related documents, click the document links in the Documents column for Submission Number 1.
3.
To edit amendment submission details, in the Action column, click the Edit icon.
The Edit Trial History page appears.
4.
In the Amendment Reason Code drop-down list, select a valid reason code. The following table lists valid reason codes.

5.
To change the amendment order, type the appropriate number in the Amendment Number field.
6.
In the Amendment Date field type or select the amendment date. See Selecting and Entering Dates.
The amendment date is generally entered by the trial submitter.
7.
Click Save.
How to Validate Trial Updates
1.
In the Navigation pane, click Trial History.
The Trial History Information page displays the Submission tab by default.
2.
Click the Updates tab.
The Updates tab displays records of any prior updates that have been validated to date. It also displays records for updates that have yet to be validated.
3.
To validate a pending trial update, click Validate, and follow the instructions for validating trials in Validating Trials.
