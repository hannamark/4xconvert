Abstracting Human Subject Review Information
The information you are required to provide depends on the Board approval status and other indicators, as noted in the instructions that follow.
How to Abstract NCI Specific Information
1.
On the navigation sidebar, click Human Subject Review.
The Human Subject Safety Information section appears.
2.
In the Board Approval Status drop-down list, select the current state of the trial with respect to Investigational Review Board (IRB) approval. Table 3.3The following table lists valid methods.
Status
Definition
Submitted, Approved
IRB approval has been requested and obtained
Submitted, Exempt
IRB has granted an exemption in response to the approval request
Submission Not Required
Study does not require human subjects review
3.
Indicate whether the Board is affiliated with an organization, for example, with the National Institutes of Health, by selecting either Yes or No.
4.
If you selected Submitted, Approved in the Board Approval Status drop-down list, type the number assigned by the human subjects review board upon approval of the protocol in the Board Approval Number field.
5.
If you selected Submitted, Approved or Submitted, Exempt in the Board Approval Status drop-down list, you must select the name of the Board from the Board Name drop-down list. If the name is not listed, search for or register a Board by following the instructions in Chapter 5, Working With Organization Records,.
6.
If you provided a Board name in the previous step, you must also provide the Board’s contact information. If you selected the Board name from the drop-down list or by searching for it, the contact information is displayed automatically in the fields provided.
7.
If you selected Submitted, Approved or Submitted, Exempt in the Board Approval Status drop-down list, you must select the official name of organizational affiliation of the approving human subjects review board from the Board Affiliation drop-down list.
8.
To save the details you have abstracted, click Save.
9.
Do one of the following to continue:
•
To review the data you provided on the previous page, click Back.
- or -
•
To proceed with the abstraction and move to the next page, click Next.
