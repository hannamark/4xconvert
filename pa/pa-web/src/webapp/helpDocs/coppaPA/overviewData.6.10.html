Editing On-hold Records
You can edit a trial that is currently on hold to indicate that the trial has resumed by entering an off-hold date. You can not make any other changes to the record.
Before you begin abstracting any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data. For instructions, see Checking In and Checking Out Trials.
How to Edit an On-hold Record
1.
Select a trial that is currently on hold.
2.
In the navigation pane, click On-hold Information.
The On-hold page appears
3.
In the Action column, click the Edit icon.
The On-hold Information page appears.
4.
If the trial has continued and is no longer on hold, enter the date on which the trial resumed in the Off-hold Date field. See Selecting and Entering Dates.
5.
Click Save.
