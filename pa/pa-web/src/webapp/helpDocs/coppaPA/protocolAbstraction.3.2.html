Searching For Trials
You can search for—and abstract or edit—any registered trial that has been validated successfullly. You can search for a particular trial to work on, or you can list all registered trials that are available for abstraction.
How to Search for Trials
1.
If the Trial Search page is not already displayed, on the PA navigation pane on the left side of the page, click Trials Search.
The Trial Search page appears.
If you do not see the search criteria, be sure that the Display Search Fields (outlined in red in 3.1) is selected.
2.
Do one of the following to search for and select available trials in the system:
•
To list all trials, leave all fields blank, and then click Search.
- or -
•
To search for a particular trial or set of trials, continue with the next steps.
3.
Type or select any of the search criteria in the following table as appropriate. You must select at least one criterion.

Type the identifier that NCI assigned to the trial when it was registered.

Type the numeric or alphanumeric identifier assigned to the study by the lead organization. Inter-Group trials use the lead Groups trial number.
