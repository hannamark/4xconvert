Abstracting Contacts
You must link at least one Participating Site to the trial before abstracting associated contact information.
How to Abstract Contact Information
1.
On the navigation pane, click Participating Sites.
The Participating Sites Page appears.
2.
Click Add.
The Participating Sites tab appears.
3.
Click the Contacts tab
The Participating Sites page displays the Contacts tab.
Note:
If a primary investigator has been designated as the primary contact (on the Investigators tab), the investigator's name is displayed automatically on the Contact tab.
4.
Next to the First Name field, click Look Up and search for the contact person’s name by following the instructions in Looking Up Registered Persons.
The person’s name you selected appears in the Name fields on the Contact tab.
5.
In the Phone Number field, type the contact’s primary telephone number (as 123-456-7890), including an extension if provided.
6.
In the Email Address field, type the contact’s primary email address.
7.
Click Save.
