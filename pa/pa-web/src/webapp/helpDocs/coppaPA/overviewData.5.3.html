Abstracting Trial History Information
The Trial History page displays information about all existing trial submissions except those that have been rejected. This page provides the means to validate and abstract amendment-specific information, and to review all submission-specific documents.
Before you begin abstracting any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data. For instructions, see Checking In and Checking Out Trials.
When an abstractor rejects an amendment, the system deletes the amendment information and sends an email message to the submitter that contains the reason for rejection.
How to Abstract Trial History Information
1.
In the navigation pane, click Trial History.
The Trial History Information page displays the submission history and provides read-only access to the original and amendment-specific documents.
2.
To review the original trial-related documents, click the document links in the Documents column for Submission Number 1.
3.
To edit amendment submission details, in the Action column, click the Edit icon ().
4.
The Edit Trial History page appears.
5.
In the Amendment Reason Code drop-down list, select a valid reason code. The following table lists valid reason codes.

6.
To change the amendment order, type the appropriate number in the Amendment Number field.
7.
In the Amendment Date field type the date, or click the calendar icon to select the amendment date.
The amendment date is generally entered by the trial submitter.
8.
Click Save.
