Reviewing Trial Documents
A Protocol and IRB document are associated with each trial. Ensure that all required documents have been submitted and that these documents include information required for abstracting trials. For example, the informed consent document must be included in the protocol document or submitted in a separate file. A participating sites document must be submitted for a multi-site trial.
You can add, edit, or delete documents that are not required. You only can replace the required ones.
How to Review Trial Documents
1.
In the navigation pane, click Trial Documents.
The Trial Documents page appears.
2.
Do one of the following to continue:
•
To add a new document, click Add and follow the instructions provided in Adding Trial-Related Documents.
•
To modify or replace an existing document, click the Edit icon () and follow the instructions provided in Editing and Replacing Documents.
•
To delete a document — other than an Protocol or IRB Approval document, click the Delete icon () in the Delete column and follow the instructions provided in Deleting Trial-Related Documents.
There may be more than one record in the table. Be sure you select the Edit or Delete icon in the same row as the record you want to modify.
3.
Proceed to Accepting and Rejecting Trials.
