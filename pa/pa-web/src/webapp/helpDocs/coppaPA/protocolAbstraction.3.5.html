Abstracting Administrative Data
You can modify or add most trial-related administrative information except identifiers that the system generates, such as NCI Trial Identifier.
Administrative data for trials comprises the following information:
•
Trial details
•
General trial details
•
NCI-specific information
•
Regulatory information
•
Human subject review
•
Trial IND/IDE
•
Trial status
•
Trial funding
•
Participating sites
•
Testing sites
•
Collaborators
•
Trial-related documentation
The navigation sidebar expands so that you can navigate easily to the specific types of data you would like to add or edit, as shown below.
The trial identification details remain displayed at the top of each page as you navigate through the Administrative Data section. This serves as a visual reminder of the trial you are abstracting or editing.
