Groups
Be able to abstract trial groups and assign interventions to groups (link intervention to group, unlink intervention from group).
Constraint/Rule:
1. This use case is applicable to observational studies only.
2. Observational study must have at least one group. This rule is verified at abstraction comple­tion.
3. Interventions must be added to trials before they can be assigned to groups.
4. Group information can be deleted. Deletion confirmation is required before issuing group dele­tion.
5. Intervention assignment must be deleted in case of corresponding group deletion or interven­tion deletion (referential integrity via FK, cascade deletion).
6. Old intervention assignment must be deleted if group is edited and interventions assignment has changed.
