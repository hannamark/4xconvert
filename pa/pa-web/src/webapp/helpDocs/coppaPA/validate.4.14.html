Validating the Other Identifiers Section
Review the identifiers submitted with the trial and make any changes as necessary. You can record additional trial identifiers such as unique identifiers from other registries, NIH grant numbers, or protocol numbers assigned by the Review Board. If the ID submitted with the trial is incorrect, delete the ID and add the correct one.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Validate the Other Identifiers Section
1.
To add an identifier to the trial, in the Other Trial Identifier field, type an additional identifier, and then click Add Other Identifier. Repeat this step for each additional identifier.
2.
The ID you added is displayed in the Other Identifier list.To delete an incorrect identifier, in the Action column, click Delete.
