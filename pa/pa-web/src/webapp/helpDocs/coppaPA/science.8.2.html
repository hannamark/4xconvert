Abstracting Trial Descriptions
Trial description information pertains to complete trials only.
Trial descriptions include the following information:
•
Brief title
•
Brief summary
Additionally, they may include a detailed description.
Before you begin abstracting or validating any trial, be sure you check it out first. Otherwise you will not be able to modify any trial data.
How to Abstract Trial Descriptions
1.
On the navigation pane, click Trial Description.
The Trial Description page appears.
2.
Enter the appropriate information in the text fields. Items with an asterisk are required fields. The Primary, Secondary, and Tertiary objective fields are read-only.
Trial Description
Instruction
Brief Title*
Type a title that summarizes the purpose of the trial.
Brief Summary*
Type a summary description of the trial.
Detailed Description
Optionally, type all the information that accurately describes the trial in detail.
3.
Click Save.
