Validating Amended Trial Submissions
There is little difference between validating an original submission and an amended version other than the trial’s history and access to additional trial-related documents.
For amended trials, you can change additional elements specific to the amendment as follows:
•
Amendment date and amendment number on the Trial History page. See Abstracting Trial History Information
•
Assigned Amendment Reason Code on the Trial History or Trial Validation page.
How to Change an Amendment Reason Code
1.
On the navigation pane, click Trial Validation and scroll to the bottom of the page.
2.
In the Amendment Reason Code drop-down list, select a valid reason code. The following table lists valid reason codes.
Reason Code
Definition
Administrative
Amendment submitted for errors or changes to non-scientific data
Example: A lead organization’s address changes
Scientific
Amendment submitted for errors or changes to the trial design
Example: A trial advances from Phase 1 to Phase 2
Both
Amendment submitted for both administrative and scientific reasons
3.
Click Save.
4.
Proceed to Accepting and Rejecting Trials.
