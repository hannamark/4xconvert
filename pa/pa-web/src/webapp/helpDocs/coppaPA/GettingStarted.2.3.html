Logging In to the NCI Clinical Protocol Abstraction Portal
Before you begin, you must request authorization to abstract trials. To request authorization, submit a request to the Help Desk via email addressed to: ncicb@pop.nci.nih.gov.
Once you register as a NCI Clinical Protocol Abstraction Portal abstractor, you can use the NCI Clinical Protocol Abstraction Portal to validate and abstract clinical trials.
How to Log In to the NCI Clinical Protocol Abstraction Portal
1.
Type your username and password in the fields provided.
2.
Click Login.
The system logs you out if it detects that you have not used the application for 30 minutes at any point in the abstraction workflow.
