This application takes XML dump of PDQ data and generates DML to load CTRP database.

1.  Download file from ftp://cipsftp.nci.nih.gov/Terminology using the username = scenpro.
2.  Extract contents into pdq/Terminology directory.
3.  Create a build.properties file useing the template.
4.  Run program.
5.  After testing the generated scripts move them to the scripts directory, appending an 
        appropriate sn and check in to svn.
