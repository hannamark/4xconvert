Viewing Trial Details
To view details for a given clinical trial listed on a search results page, click its associated NCI Trial Identifier hypertext link.
The Trial Details page displays the metadata as entered by a trial submitter. Refer to Metadata Definitions for a description of the metadata.
Responsible party, IND/IDE, NIH grant information and trial-related documents are only displayed for the private trials.
