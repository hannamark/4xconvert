Preparing the Trial Data File for Abbreviated Trials
The CTRP website provides a batch upload template, valid data values, data definitions, examples, and rules for registering batches of abbreviated trials.
Ηοω το Prepare the Trial Data File
1.
In Microsoft Excel, list the trial elements required for registration in the order specified in the Registry_Abbreviated_Trial_Template.xls file.
2.
Email the file to CTRO at ncictro@mail.nih.gov as an attachment.
CTRO staff process your file and register your trials.
Because the CTRO staff submit your trials, they maintain trial ownership by default.
To assume ownership of the trials, send a request for trial ownership to the CTRO along with the .xls file.
