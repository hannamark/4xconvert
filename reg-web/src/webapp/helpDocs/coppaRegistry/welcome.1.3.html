Using Navigation Tools
•
The breadcrumb trail of topics at the top of the page, for example, Using Online Help > Using Navigation Tools, shows the path you took to get to this page. Click a breadcrumb link to go back to a previous page you’ve viewed.
•
Follow blue hypertext links or the Related Topics buttons on the help page to open closely related topics. You can also click at the top right of the help page to see the related topics.
•
To expand or narrow the width of the table of contents or topic pane, click and drag the divider between them.
