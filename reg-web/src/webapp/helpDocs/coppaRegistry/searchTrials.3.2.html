Conducting Searches in the CTRP Registration Site
The CTRP Registration Site enables you to search for trials, persons, and organizations that have been registered in the CTRP system. In each case you are presented with a pre-defined set of search criteria that is appropriate for the type of search you are conducting. For example, when you search for a trial, the set of criteria includes Title, Trial Phase, and Purpose, whereas a set of criteria for searching for a person will include Last name and First Name. Instructions for searching for persons and organizations are provided in Appendix A, Working With Persons and Organizations.
All criteria are presented as fields in which you type attributes, or as drop-down lists from which you select them. You can run your search once you have selected or typed an attribute for at least one criteria.
You can type a series of characters, rather than a complete word or phrase, in any of the search fields. Because the system adds wildcards for you, do not type wildcard symbols in the search fields.
If the trial you searched for is not listed in the search results table, you may have searched too narrowly (that is, you may have provided too much information about the trial). To broaden your search so that more trials are listed in the search results, delete one or more of your criteria. For example, if you searched by both Phase and Principal Investigator in your original search, you may want to search by Principal Investigator alone.
If the list of results is very long and contains many trials that are similar to the one you are searching for, you can narrow your search by providing more information. To narrow your search so that fewer trials are listed in the search results, provide more information about the trial. For example, if you searched by Principal Investigator only in your original search, you may want to search by Phase as well.
