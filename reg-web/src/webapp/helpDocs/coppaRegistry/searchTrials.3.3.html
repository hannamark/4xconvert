Searching For Trials
You can retrieve existing interventional trials through the CTRP Registration Site once you have registered for an account. See Creating and Updating CTRP Registration Site Accounts
You can search for the following categories of trials:
•
Public trials. All trials registered with the CTRP from all organizations/accounts. Use the Search All Trials feature.
•
Private trials. Trials that you have submitted. Use the Search My Trials feature.
•
Partial submissions. Trials that you have saved as drafts but have not submitted. Use the Search Saved Drafts feature.
The search feature you choose determines which categories of trials will be returned, as indicated by the white rectangles in the figure below. The black rectangles with rounded corners indicate what actions you can perform on the different types of search result records. Refer to Viewing Trial Details for rules that determine which trial details are displayed.
All registered users can search trials with the “Accepted” and subsequent processing status. Additionally, you can search trials that you registered or own which have not been validated. These trials are indicated by the “Submitted” status.
How to Search for Existing Trials
1.
On the navigation pane on the left side of the page, click Search Trials.
The Search Trials page appears.
2.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. Items with an asterisk are required fields.
If you are searching for a saved draft, search by Phase, Purpose, Title, or Identifier only. Because the system adds wildcards for you, do not type wildcard symbols in the search fields.
