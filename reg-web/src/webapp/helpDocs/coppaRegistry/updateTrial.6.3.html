Updating Trials
As trial owner, you can update most information included with the original trial submission, including the following:
•
Other Identifiers
•
Other identifier (add/modify existing)
•
NCT Number
•
CTP/DCP Identifier
•
Purpose
•
Purpose description
•
Sponsor/Responsible Party
•
Responsible Party (change selection)
•
Responsible Party Email Address
•
Responsible Party Phone Number
•
Summary 4 information (for trials at NCI-designated cancer centers)
•
Summary 4 Funding Sponsor
•
Program Code
•
NIH Grant information (for NIH-funded trials)
•
Add new unique grant (do not submit duplicates)
•
Status Dates (all fields)
Changing the overall trial status must reflect changes to the trial status at the site. For example, if you change the overall status from “Approved” to “Active”, you must change the recruitment status from “not yet recruiting” to “recruiting.”
•
IND/IDE information (for applicable trials)
•
Add new unique IND/IDE (do not submit duplicates)
•
Regulatory Information
•
Participating sites (all fields)
•
Site recruitment status and associated date and site program code for abstracted trial sites. See Current trial status mappings and descriptions for valid recruitment statuses, their definitions in the CTRP Registration Site, and equivalents in ClinicalTrials.gov.
•
Collaborator’s functional role (funding source)
•
IRB approval documents (submit new document)

1.
On the navigation pane on the left side of the page, click Search Trials.
The Search Trials page appears
2.
Click Search My Trials.
The Search Trials page refreshes and displays a list of search resultsFor information about navigating the search results list, see Viewing Trial Details.
3.
Locate the trial you want to update, and then click the Update link in the corresponding column.
The Update Trial page displays the data currently registered with CTRP
4.
Make changes to the fields as necessary. Instructions for recording each of the fields are provided in Registering New Trials and Registering New Industrial Trials.
5.
If appropriate, upload the documents for IRB approval.
If you upload an IRB document, the CTRO reviews the updated record you submit and makes changes to the record as necessary. For example, if you upload an IRB document for a trial currently in the “In Review” status, the CTRO updates the IRB information section of the trial record (e.g., IRB status and approval number).
6.
To review the information you provided, click Review Trial.
The system checks the updated information for errors, and displays the results at the top of the Update Trial page.
7.
If necessary, correct any errors, and click Review Trial. Repeat this cycle until your update is error-free.
8.
Submit the trial update.
The system sends you an update notification whenever you update accepted trials.
