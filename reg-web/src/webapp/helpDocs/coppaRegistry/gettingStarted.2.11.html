Using the Proprietary Trial Template
The proprietary trial template is designed primarily for submitting multiple industry-lead trials with contractual obligations that restrict sharing of their protocol documents.
You must email the completed trial Proprietary template to the CTRO staff at mailto:ncictro@mail.nih.gov. The CTRO will then register your trials using the data you provided.
You can use the same template to add participating site information to a proprietary trial that a different participating site has already registered.
