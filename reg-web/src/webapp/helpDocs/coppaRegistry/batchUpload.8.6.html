Trial Specification Rules For Proprietary Trials
The Registry_Proprietary_Trial_Template.xls file lists all trial data required for registering multiple trials in batches.
Examples of rules for submitting specified data include the following:
•
Maintain the order in which the data appear in the .xls file
•
Uniquely identify each trial
•
If you provide an NCT trial number, you do not have to provide the following data:
•
primary purpose
•
phase
If the CTRO identifies a trial as a duplicate to an existing trial, you can request that they add your organization information to the existing trial as a participating site instead of creating a duplicate.To access the Registry_Proprietary_Trial_Template.xls, see Downloading Trial Registration Templates.
