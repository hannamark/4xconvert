Granting and Revoking Administrative Authority
 
As a site administrator, you can grant and revoke administrative rights to other users in your organization. Site administrators are the only CTRP Registration Site users who have access to the Site Administration menus on the navigation pane. No other user will see these menus.
Site administrators can view only those trials within their own organization.
How to Grant and Revoke Site Administrator Rights
1.
On the navigation pane on the left side of the page, click Site Administration. The Site Administration page displays the current administrators if you have granted administrator rights previously.To search for a user you want to promote to site administrator, search for the user by first name, last name, or email address, and then click Search.
The userâ€™s name appears in the Search Results list.
2.
To promote the user to site administrator, select the check box in the Allow Site Admin column.
As a site administrator you can revoke your own administrative rights. Use caution if you do so because you can not promote yourself thereafter.
3.
To revoke administrative access, locate the user in the Search Results section, and clear the check box under the Allow Site Admin column.
4.
Click Save.
