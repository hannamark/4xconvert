Completing the Trial Details Section for Proprietary Trials
You must complete all fields in the Trial Details section.
How to Complete the Trial Details Section
1.
Type the Title in the field provided. You can use a maximum of 4000 characters.
For example:
“Study of Recombinant Vaccinia Virus That Expresses Prostate Specific Antigen in Metastatic Adenocarcinoma of the Prostate”
1.
The Interventional Trial Type is pre-selected.
Currently you can register interventional trials only. Future releases of this produce will enable you to register observational trials as well.
2.
From the Purpose drop-down list, select the purpose of the trial. See Trial purpose definitions for valid values.
3.
From the Phase drop-down list, select the trial phase. See Trial phase definitions.
4.
Look up the Principal Investigator and select the appropriate name from the list of search results. If your trial’s principal investigator’s name is not listed, you can register it in the system at this point. To search for and register an investigator, follow the instructions in Searching for Principal Investigators and Adding Principal Investigators.
