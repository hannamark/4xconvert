Amending Verified Trials
In your role as trial owner (original submitter or current owner), you can amend only non-proprietary trials. The trials you own are listed when you use the Search My Trials feature. See Searching For Trials.
You are required to provide information for all fields marked with an asterisk (*).
How to Amend Verified Trials
1.
On the navigation pane on the left side of the page, click Search Trials.
The Search Trials page appears.
If you know the NCI trial number of your validated trial that you want to amend, in the Identifier Type field, select NCI, and then type the trial number in the Identifier field.
2.
Click Search My Trials.
The Search Trials page refreshes and displays a list of search results.
For information about navigating the search results list, see Navigating Through the Search Results List.
3.
Locate the trial you want to amend, and then click the Amend link in the Amend column.
The Amendment Trial page displays the data currently registered with the CTRP.
4.
If you require an XML document to register your trial with ClinicalTrials.gov, next to XML required?, select the Yes option. If you are not going to submit your trial to ClinicalTrials.gov, select the No option. The option you select here will dictate which other sections you will be required to complete. For example, if you select No, you will not be required to complete responsible party and regulatory information.If you select Yes, NCI will be added as a collaborator to the Funding Source.
5.
In the Amendment Number field, type the number as recorded in the system (as designated by the lead organization) or in the amended protocol document.
Type the Amendment Number in alpha-numeric characters, dashes, and other special characters as appropriate.
6.
In the Amendment Date field, type the date on which the trial was amended using the mm/dd/yyyy format, or, click the calendar icon () and select the date from the calendar.
7.
Make changes in the rest of the fields as per the amended protocol.
8.
Upload the amendment-specific documents.
9.
Review the amendment.
10.
Submit the amended trial to the CTRP.
Instructions for providing/editing most of the trial amendment details are provided in Registering Non-Proprietary Trials. Amendment-specific rules, values, etc., are provided for each section of the Amendment Trial page.
