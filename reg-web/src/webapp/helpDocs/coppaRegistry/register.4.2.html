Adding New Trials
NCI Clinical Trials Reporting Portal provides a user-friendly interface through which you can submit new trials.
Before you begin to register a trial, ensure that the trial does not exist in the system already. You can do this by searching for trials using any of the criteria as per the instructions in Searching For Trials.
You are required to provide information for all fields marked with an asterisk (*).

•

How to Submit a Trial:
1.
On the navigation pane on the left side of the page, click Add a Trial.
The Add a Trial page appears.
2.
Type the appropriate information in the text fields, or select options from the drop-down lists as appropriate.
3.
Click Submit Trial.
After submission, no other users can see the trial information you provided until the information has been validated. Once validated, the trial you submitted is ready for abstraction by another specialist.
The system generates an e-mail message to acknowledge that the trial has been submitted. It generates a similar message following successful abstraction of a validated trial. If the protocol is rejected at validation, the system alerts you via a rejection message.
