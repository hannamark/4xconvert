Preparing Trial-Related Documents
You must provide trial-related documents as .zip files.
How to Prepare the Trial-Related Documents
1.
Rename each file. Add the unique trial identifier (for example, your cancer center unique trial identifier) to the beginning of the file name.
2.
Ensure that no two documents for the same trial share the same file name.
3.
List the trial-related document file names for each trial in the trial data file (.xls file). You can list up to 5 files per trial record.
4.
Zip all the trial documents. Do not include path names in the .zip file name.
