What’s New in This Release
CTRP Registration Site version 3.2 has the following new features:
•
“Save as Draft” feature allows you to complete registration of a given trial over multiple CTRP Registration Site sessions
•
New user interface that allows you to do the following:
•
register non-proprietary trials that do not require an XML document for submission to ClinicalTrials.gov
•
record Regulatory information details
•
update proprietary trials via the CTRP Registration Site or caGrid service
•
indicate which trials will be submitted to ClinicalTrials.gov, so that you can be sure to fulfill the appropriate requirements (an .xml file is generated for trials that will be submitted to ClinicalTrials.gov)
•
search for trials using CTEP, DCP, and other trial identifiers
•
request administrative access to trials in your organization
•
Greater security
•
Submitting site users can request the Trial Summary Report (TSR) and XML documents on demand when logged in to the application
•
Site administration feature allows site administrators to grant trial ownership to multiple users in an organization
•
Improved document attachment process streamlines the trial update process
•
Improved batch template streamlines batch upload
•
NCI is added as a trial collaborator to all trials by default
