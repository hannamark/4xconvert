What’s New in This Release
CTRP Registration Site version 3.2 has the following new features:
•
Ability to complete registration of a given trial over multiple CTRP Registration Site sessions
•
New user interface for registering non-proprietary trials that do not require an XML document for submission to ClinicalTrials.gov
•
Greater security via account creation process
•
Ability for the submitting site users to request the Trial Summary Report (TSR) and XML documents on demand when logged in to the application
•
Ability to record regulatory information details
•
Ability to grant access to trials to multiple users in an organization
•
Ability to update proprietary trials via the CTRP Registration Site or caGrid service
•
Ability to indicate which trials will be submitted to ClinicalTrials.gov, so that you can be sure to fulfill all requirements
•
Improved document attachment process means that it is no longer necessary to re-attach documents during updates
•
Streamlined batch template
•
Ability to search for trials using CTEP, DCP, and other trial identifiers
•
Ability to request administrative access for your organization
•
NCI is added as a trial collaborator to all trials by default
