Data Requirements for Batch Uploads
Elements that are required for single trial registration are also required for batch uploads, with the exception of person/organization attributes. The complete set of person/organization attributes for registering new persons/organizations is required for Principal Investigator, Lead Organization, Sponsor, and Summary 4 Sponsor/Source trial functional roles.
The Batch Upload application can only be used to submit complete trials.
Trial data in the attached files must conform to the specifications in the CTRP Registry Complete Batch Upload Template that is valid for the current release.
Detailed specifications are provided in Appendix C, Batch Upload Data Specifications.
