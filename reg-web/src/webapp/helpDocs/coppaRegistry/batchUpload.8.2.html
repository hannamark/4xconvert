Preparing the Trial Data File for Non-Proprietary Trials
The CTRP website provides a batch upload template, valid data values, data definitions, examples, and rules for registering batches of trials and trial amendments/updates. This section contains a summary of the information required for the .xls file. See Using CTRP Registration Templates and Resources
The .xls file that contains the trial data you want to register via the CTRP Registration Site’s batch upload feature, and the trial-related documents associated with it, must meet certain specifications for successful registration.
You must provide trial data in the Microsoft Excel format, .xls.
