Amending NIH Grant Details
You can create a new NIH grant record only if you can provide all of the following details:
•
Funding Mechanism
•
NIH Institution Code
•
Serial Number
•
NCI Division/Program
Instructions for completing the NIH grant section are the same for both original and amended submissions. See Completing the NIH Grant Information Section
You cannot change an existing NIH grant number.
