Working with Search Results
The system reviews, or validates each trial submitted to the system. During the validation process, the system checks for duplicate records and ensures that the submitter has provided all required information.
The search returns results and displays them according to the status of the trial at the time of your search, and the user conducting the search. The following table provides definitions for each of the states and indicates which ones will be displayed for different types of users.
Note:
Only trials that you submitted display a status in the search results list.
