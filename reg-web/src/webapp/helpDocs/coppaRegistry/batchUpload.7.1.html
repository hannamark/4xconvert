Batch Upload Data Specifications
The .xsl file that contains the trial data you want to register via the NCI Clinical Trials Reporting Program’s batch upload feature, and the trial-related documents associated with it, must meet certain specifications for successful registration.
This section describes how to prepare your trial documents. It also provides data specifications for the trial data.
