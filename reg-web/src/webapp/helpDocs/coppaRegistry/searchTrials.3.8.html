Requesting Trial Summary Reports and XML Documents
You can request a copy of a Trial Summary Report (TSR) or the XML document for a trial you own if it meets the following criteria:
•
The trial is private and non-proprietary
•
Requires submission to ClinicalTrials.gov (for XML documents)
•
Has a processing status “Abstraction Verified” or subsequent status

1.
On the navigation pane on the left side of the page, click Search Trials to locate the trial that you want to request a TSR or XML document for.
2.
The trial you searched for appears in the Submitted Clinical Trials Search Results table.
3.
In the Action column, click Send XML or Send TSR.
4.
The system sends the document you requested to the trial’s owner(s) and site administrator(s).
