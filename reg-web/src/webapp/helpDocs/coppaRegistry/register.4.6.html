Completing the Trial Identifiers Section
You must provide a Lead Organization trial identifier for each trial you register. Optionally, provide an NCT number in addition.
How to Trial IdentificationComplete the Trial Identifiers Section
1.
Indicate whether or not you require an XML document for registration with ClinicalTrials.gov. The default is Yes. If you select No, you will not be asked to provide regulatory and responsible party information.
2.
Type the Lead Organization Trial Identifier in the field provided, or for Inter-Group trials, type the Lead Groupâ€™s trial number.
The Trial Identifier must be exactly the same as it appears in the protocol document.
For example:
NSABP-B-40
For multi-site trials that have no assigned single center, use the protocol ID.
3.
Optionally, type the NCT number in the field provided. This number is assigned to trials that have been submitted to ClinicalTrials.gov previously.
4.
If you want to record additional trial identifiers such as unique identifiers from other registries, NIH grant numbers, or protocol numbers assigned by the Review Board, in the Other Trial Identifier field, type an additional identifier, and then click Add Other Identifier. Repeat this step for each additional identifier.
5.
The ID you added is displayed in the Other Identifier list.To delete an identifier you have added, in the Action column, click Delete.
