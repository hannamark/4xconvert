What’s New in This Release
CTRP Registration Site version 3.3 has the following new features:
•
Users indicate whether a trial is Industrial, National, Externally Peer-Reviewed, or Institutional as the first step in the registration process. The chosen value determines the trial category and submission workflow.
In previous versions, trial categorization was based on protocol document availability (i.e. whether a protocol was proprietary or non-proprietary). In this release, the trial categorization is based on the Summary 4 Funding Sponsor Type as follows:
•
Abbreviated trials. Trials with Summary 4 Funding Sponsor Type “Industrial.”In earlier releases, these trials were referred to as “Proprietary”.
•
Complete trials. Trials with Summary 4 Funding Sponsor Type “National”, “Externally Peer-Reviewed”, and “Institutional”. In earlier releases, these trials were referred to as “Non-Proprietary”.
•
User can indicate whether a trial IND/IDE is exempt from reporting to ClinicalTrials.gov. FDA-regulated intervention rules do not apply to trials with IND/IDE exemptions.
•
The valid values available for Trial Phase are the same as the ClinicalTrials.gov values. You can now indicate whether a trial is a Pilot when you select the Phase value “N/A”.
•
The valid values available for Primary Purpose are the same as the ClinicalTrials.gov values. You can now describe the Primary Purpose in text form if you select the Purpose value “Other”.
•
Trial Summary Reports (TSRs) for registered trials, including trials registered prior to this release, are now available as Word and HTML documents instead of PDFs.
•
Label-level help for trial registration is now available as tool tips that pop up upon hovering your mouse over form field labels.
•
Summary 4 Funding Sponsor Type and Summary 4 Funding Sponsor Organizations values are mandatory for submitting trials to CTRP.
•
You can now change the Trial Status “in-Review” to “Active”, and from “Closed to Accrual” to “Completed” when updating and amending trials.
