Creating New CTRP Registration Site Accounts
If you are new to the NCI Clinical Trials Reporting Program Trial Registration Site and you do not have a caGrid or NCI account, request aCTRP Registration Site account via the CTRP Registration Site account creation feature.
How to Create a New CTRP Registration Site Account
1.
Navigate to the CTRP Registration Site home page at http://trials.nci.nih.gov/registration
2.
Click Create an account.
The Create Account page appears.
3.
In the Email Address field, type the email address you will use for registering trials (this should be your business email address), and then click Submit.
The system sends you a link to activate your account.
4.
Open the confirmation email and click the embedded link to the My Account page.
The My Account page appears. The email field is pre-populated with the email address you provided
 
5.
Select or enter the appropriate information in the text fields and drop-down lists. Fields are described in the following table. All fields are required.
Field Label
Description/Instructions
Username
Type the username you would like to use with this account.
Password
type a new password for your account. Passwords must have the following characteristics:
•
A minimum of 10, and fewer than 20 characters
•
At least one upper case letter, one lower case letter, one digit, and one non-alphanumeric symbol
•
No dictionary words
You can use the account you create here for all other caGrid-enabled applications.
Re-type password
Type your new password again. This ensures that you have typed your password correctly the first time.
Your Account Profile section
Complete the remaining personal information fields. Provide your professional information only.
Contact information is required for internal administrative use only. Your information is not revealed to the public.
Organization Affiliation
Click Look Up and select the appropriate organization from the list of search results. If your trial’s affiliated organization is not listed, you can register it in the system at this point. To search for and register an organization, follow the instructions in Looking Up Registered Persons and Adding Persons to CTRP.
Once organization affiliation is assigned, if no one in your organization has site administrative (admin) access, the My Account page displays a check box that enables you to request admin access. Otherwise the View Admins link that enables you to display the name of the current admins appears.1
Request for Admin Access
This check-box is displayed if no one in your organization has site administrative (admin) access.
Optionally, to request administrative rights for your organization, select the check box.
You will receive an email message from the CTRO once your request has been approved or rejected. For more information about administrative access, see Managing Organizational Access.
PRS Organization Name
Type the full name of the organization as it appears in the ClinicalTrials.gov Protocol Registration System account list at http://prsinfo.clinicaltrials.gov/prs_organizations.html.
The PRS organization name is required for uploading trial records to ClinicalTrials.gov via a system-generated .xml file. The PRS organization name you include in your profile is included in that .xml file. This precludes having to update the PRS name in the .xml file. Therefore it is very important for you to update your account whenever there is a change in PRS.
Submit
Click to register your information with CTRP.
The system sends you an account confirmation email message.
1
The link appears only if the organization you are affiliated with already has one or more regis­tered admins.
