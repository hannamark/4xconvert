NCI Clinical Trials Reporting Program Registration Site Online Help
Welcome to the NCI Clinical Trials Reporting Program Registration Site Online Help. To start using the CTRP Registration Site, select from the following topics:
•
Getting Started introduces you to the CTRP Registration Site and provides instructions for registering for an account and for logging in to the system.
•
Searching For Trials describes how to search for trials in the NCI Clinical Trials Reporting Program Registration Site.
•
Registering New Trials describes how to submit, or register, trials using the NCI Clinical Trials Reporting Program Registration Site.
•
Amending Registered Trials describes how to make amendments to trials currently registered and verified in the CTRP, and how to submit the amended trials using the NCI Clinical Trials Reporting Program Registration Site.
•
Managing Your Account provides instructions for modifying your NCI Clinical Trials Reporting Program Registration Site account, and for resetting your password.
•
Metadata Definitions provides definitions and examples of the metadata associated with clinical trials.
•
Batch Upload Data Specifications describes how to prepare your trial documents. It also provides data specifications for the trial data.
•
Participating Sites Document Specifications provides the specifications—rules, formats, requirements, etc.—for Participating Sites documents.
•
Glossary provides definitions of acronyms, abbreviations, and terminology.
•
Application Support provides contact information for general application support.
To learn more about each topic, browse through the online help or use the table of contents, index, or search features of the help. For more information, see Using the Online Help.
You can also find comprehensive procedures for all NCI Clinical Trials Reporting Program Registration Site tasks in the NCI Clinical Trials Reporting Program Registration Site User’s Guide.
