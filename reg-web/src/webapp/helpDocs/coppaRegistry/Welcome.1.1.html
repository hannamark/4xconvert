NCI Clinical Trials Reporting Program Online Help
Welcome to the NCI Clinical Trials Reporting Program Online Help. To start using the NCI Clinical Trials Reporting Program, select from the following topics:
•
Getting Started introduces you to the NCI Clinical Trials Reporting Program and provides instructions for registering for an account and for logging in to the system.
•
Searching For Trials describes how to search for, submit, and manage trials in the NCI Clinical Trials Reporting Program.
•
Registering New Trials describes how to submit, or register, trials using the NCI Clinical Trials Reporting Program.
•
Managing Your Account provides instructions for modifying your NCI Clinical Trials Reporting Program account, and for resetting your password.
•
Metadata Definitions provides definitions and examples of the metadata associated with clinical trials.
•
Glossary provides definitions of acronyms, abbreviations, and terminology.
•
Application Support provides contact information for general application support.
To learn more about each topic, browse through the online help or use the table of contents, index, or search features of the help. For more information, see Using the Online Help.
You can also find comprehensive procedures for all NCI Clinical Trials Reporting Program tasks in the NCI Clinical Trials Reporting Program User’s Guide.
