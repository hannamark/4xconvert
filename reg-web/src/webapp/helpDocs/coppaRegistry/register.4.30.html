Registering Multiple Trials in a Batch
Before you begin, you must request authorization from the CTRP to upload batches of trials. To request authorization, submit a request to the Help Desk via email addressed to ncicb@pop.nci.nih.gov, or call 888-478-4423.
As an authorized CTRP submitter you can use the CTRP Registration Site’s batch upload feature to register multiple new non-proprietary and proprietary trials that were conducted at a given site, and to update and/or amend them as necessary.
The CTRP provides you with the link to the batch upload website when it issues authorization to upload batches.
In the current release, you can supply a maximum of 5 trial documents.
You must upload each of the following types of files when you register multiple trials:
•
Data documents – Documents that contain all the requisite information about the protocol. See Data Requirements for Batch Uploads and Appendix A, Batch Upload Data Specifications.
•
Format: Microsoft Excel file ({filename}.xls)
•
Trial-related documents – Protocol and IRB documents, among others.
•
Format: compressed Word files ({filename}.zip)
Currently you are required to supply your documents as compressed Microsoft Word (.doc, .docx, or .docm), Adobe PDF, Microsoft Excel (.xls, .xlsx, .xlsm, or .xlsb), and/or WordPerfect files.
How to Upload a Batch of Trials
1.
Navigate to the batch upload URL that you received from the CTRO.
The Batch Upload page appears. All fields are required.
2.
In the Organization Name field, type the name of the organization associated with the trials you want to register.
3.
Beside the Trial Data field, click Browse and navigate to the .xls file that contains all the trial data. See Data Requirements for Batch Uploads
4.
Beside the Documents Zip field, click Browse and navigate to the .zip file that contains all the trial-related documents.
5.
Click Upload Trial.
The batch upload program generates a report after processing the batch data and emails it to the submitter. The report includes a brief summary and the detailed status of each trial.
