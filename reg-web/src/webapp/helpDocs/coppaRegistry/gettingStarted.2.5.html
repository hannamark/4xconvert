Creating CTRP Registration Site Accounts
To search for and submit individual clinical trial protocol details, you must have an account in the NCI Clinical Trials Reporting Program Trial Registration Site. You can create an account using the application’s account creation feature.
The way in which you create a new account in the CTRP Registration Site depends on whether you have a current NIH or caGrid account, as follows:
•
New users who do not have a caGrid or NIH account request caGrid credentials via the CTRP Registration Site. Follow the instructions in Creating New CTRP Registration Site Accounts.
•
New users who have a caGrid or NIH account use their current credentials to register for an account via the CTRP Registration Site. Follow the instructions in Creating New CTRP Registration Site Accounts Using caGrid or NCI Credentials.
New users must have access to a valid email address to create an account.
Additionally, you can request authorization from the CTRO to use the batch upload feature to register, update, and amend multiple new trials that were conducted at a given site. Follow the instructions in Registering Multiple Trials in a Batch.
